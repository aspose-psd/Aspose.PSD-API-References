## JpegLsInterleaveMode enumeration

Определяет режим чередования для многокомпонентных (цветных) пиксельных данных.

```csharp
public enum JpegLsInterleaveMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| None | `0` | Данные кодируются и сохраняются как компонент для компонента: RRRGGGBBB. |
| Line | `1` | Режим чередования - построчный. Перед переходом к следующей строке кодируется полная строка каждого компонента. |
| Sample | `2` | Данные кодируются и сохраняются по семплам. Для цветных изображений это такой формат, как RGBRGBRGB. |
