## Layer.IsVisibleInGroup property

Получает значение, указывающее, виден ли этот экземпляр в группе (если слой не входит в группу, это означает корневую группу).

```csharp
public virtual bool IsVisibleInGroup { get; }
```

### Стоимость имущества

`истинный` если этот экземпляр виден в группе; в противном случае,`ЛОЖЬ` .
