## GlobalLayerMaskInfo.Kind property

Получает или устанавливает вид. 0 = выбранный цвет, т. е. инвертированный; 1 = защищенный цвет; 128 = использовать значение, сохраненное для каждого слоя. Это значение является предпочтительным. Остальные значения предназначены для обратной совместимости с бета-версиями.

```csharp
public byte Kind { get; set; }
```
