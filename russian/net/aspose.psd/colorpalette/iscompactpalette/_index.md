## ColorPalette.IsCompactPalette property

Получает или задает значение, указывающее, используется ли компактная палитра.

```csharp
public bool IsCompactPalette { get; }
```

### Стоимость имущества

` true` если используется компактная палитра; в противном случае` false` .

### Примечания

Компактная палитра означает, что изображение будет содержать только указанные элементы палитры, если это возможно, или, другими словами, изображение будет более компактным и занимать меньше места; иначе будет 2^BitsPerPixel записей, и изображение будет резервировать больше места для всех возможных записей палитры. Установка этого значения в true и изменение элементов палитры может привести к снижению производительности, поскольку может произойти перемещение данных, поэтому используйте его осторожно.
