## LineCap enumeration

Определяет доступные стили заглавных букв, с которыми[`Pen`](../pen/) объект может заканчиваться строкой.

```csharp
public enum LineCap
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Flat | `0` | Определяет плоскую линию. |
| Square | `1` | Задает квадратную линию. |
| Round | `2` | Определяет круглую шапку линии. |
| Triangle | `3` | Задает треугольный конец строки. |
| NoAnchor | `16` | Не указывает привязку. |
| SquareAnchor | `17` | Указывает квадратную заглушку линии привязки. |
| RoundAnchor | `18` | Определяет круглую заглушку анкера. |
| DiamondAnchor | `19` | Задает колпачок ромбовидного анкера. |
| ArrowAnchor | `20` | Задает колпачок анкера в форме стрелки. |
| AnchorMask | `240` | Указывает пользовательский заголовок строки. |
| Custom | `255` | Определяет маску, используемую для проверки того, является ли конец строки якорным концом. |
