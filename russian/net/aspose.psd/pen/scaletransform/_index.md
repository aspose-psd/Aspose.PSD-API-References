## ScaleTransform(float, float) {#scaletransform}

Масштабирует локальное геометрическое преобразование по указанным коэффициентам. Этот метод добавляет матрицу масштабирования к преобразованию.

```csharp
public void ScaleTransform(float sx, float sy)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| sx | Single | Коэффициент масштабирования преобразования в направлении оси x. |
| sy | Single | Коэффициент масштабирования преобразования в направлении оси Y. |
