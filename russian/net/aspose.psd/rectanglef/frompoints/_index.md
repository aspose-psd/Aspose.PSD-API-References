## RectangleF.FromPoints method

Создает новый[`Rectangle`](../../rectangle/) из двух указанных точек. Две вершины созданного[`Rectangle`](../../rectangle/) будет равно переданному*point1* и*point2* . Обычно это противоположные вершины.

```csharp
public static RectangleF FromPoints(PointF point1, PointF point2)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| point1 | PointF | Первый[`Point`](../../point/) для нового прямоугольника. |
| point2 | PointF | Второй[`Point`](../../point/) для нового прямоугольника. |

### Возвращаемое значение

Недавно созданный[`Rectangle`](../../rectangle/).
