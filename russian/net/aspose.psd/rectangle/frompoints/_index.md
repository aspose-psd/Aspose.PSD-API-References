## Rectangle.FromPoints method

Создает новый[`Rectangle`](../) из двух указанных точек. Две вертикали созданного[`Rectangle`](../) будет равно переданному*point1* и*point2* . Обычно это противоположные вершины.

```csharp
public static Rectangle FromPoints(Point point1, Point point2)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| point1 | Point | Первый[`Point`](../../point/) для нового прямоугольника. |
| point2 | Point | Второй[`Point`](../../point/) для нового прямоугольника. |

### Возвращаемое значение

Недавно созданный[`Rectangle`](../).
