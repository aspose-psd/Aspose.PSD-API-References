## Rectangle Inequality operator

Проверяет, являются ли два[`Rectangle`](../) структуры отличаются расположением или размером.

```csharp
public static bool operator !=(Rectangle left, Rectangle right)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| left | Rectangle | [`Rectangle`](../) структура слева от оператора неравенства. |
| right | Rectangle | [`Rectangle`](../) структура, которая находится справа от оператора неравенства. |

### Возвращаемое значение

Этот оператор возвращает true, если любой из[`X`](../x/) ,[`Y`](../y/) ,[`Width`](../width/) или[`Height`](../height/) свойства двух[`Rectangle`](../) структуры неодинаковые; иначе ложно.
