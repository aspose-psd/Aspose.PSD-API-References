## Figure.IsClosed property

Получает или задает значение, указывающее, закрыта ли эта фигура. Замкнутая фигура будет иметь значение только в том случае, если первая и последняя фигуры являются непрерывными фигурами. В таком случае первая точка первой фигуры будет соединена прямой линией с последней точкой последней фигуры.

```csharp
public bool IsClosed { get; set; }
```

### Стоимость имущества

` True` если эта фигура закрыта; в противном случае` false` .
