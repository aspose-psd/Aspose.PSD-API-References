## IOrderedShape.IsClosed property

Получает или задает значение, указывающее, закрыта ли упорядоченная фигура. При обработке замкнутой упорядоченной формы начальная и конечная точки не имеют значения.

```csharp
public bool IsClosed { get; set; }
```

### Стоимость имущества

`Истинный` если эта упорядоченная форма замкнута; в противном случае,`ЛОЖЬ`.
