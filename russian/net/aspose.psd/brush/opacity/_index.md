## Brush.Opacity property

Получает или задает непрозрачность кисти. Значение должно быть от 0 до 1. Значение 0 означает, что кисть полностью видна, значение 1 означает, что кисть полностью непрозрачна.

```csharp
public float Opacity { get; set; }
```

### Стоимость имущества

Значение непрозрачности кисти.
