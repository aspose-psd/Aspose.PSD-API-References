## Point Inequality operator

Сравнивает два[`Point`](../) объекты. Результат указывает, являются ли значения[`X`](../x/) или[`Y`](../y/) свойства двух[`Point`](../) объекты не равны.

```csharp
public static bool operator !=(Point point1, Point point2)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| point1 | Point | Первый[`Point`](../) сравнивать. |
| point2 | Point | Второй[`Point`](../) сравнивать. |

### Возвращаемое значение

Истинно, если значения[`X`](../x/) свойства или[`Y`](../y/) свойства*point1* и*point2* различаться; в противном случае ложно.
