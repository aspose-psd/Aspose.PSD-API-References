## FontSettings.GetFontsFolders method

Получает копию массива, содержащего список папок, в которых Aspose.Words ищет шрифты TrueType.

```csharp
public static string[] GetFontsFolders()
```

### Возвращаемое значение

Копия текущих расположений шрифтов.

### Примечания

Возвращаемое значение является копией данных, которые использует Aspose.Words. Если вы измените записи в возвращаемом массиве, это не повлияет на отрисовку документа. Чтобы указать новые местоположения шрифта, используйте[`SetFontsFolders`](../setfontsfolders/) метод.
