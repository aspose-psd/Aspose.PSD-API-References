## Color Inequality operator

Проверяет, являются ли два указанных[`Color`](../) структуры разные.

```csharp
public static bool operator !=(Color left, Color right)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| left | Color | [`Color`](../) слева от оператора неравенства. |
| right | Color | [`Color`](../) то есть справа от оператора неравенства. |

### Возвращаемое значение

Правда, если два[`Color`](../) структуры разные; в противном случае ложно.
