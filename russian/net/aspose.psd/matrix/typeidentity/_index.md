## Matrix.TypeIdentity field

Преобразование идентичности — это преобразование, в котором выходные координаты всегда совпадают с входными координатами. Если это преобразование отличается от преобразования идентичности, тип будет либо константой GENERAL_TRANSFORM, либо комбинацией соответствующих битов флага для различные преобразования координат , которые выполняет это преобразование.

```csharp
public const int TypeIdentity;
```
