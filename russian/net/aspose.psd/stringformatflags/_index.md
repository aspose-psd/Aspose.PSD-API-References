## StringFormatFlags enumeration

Указывает информацию об отображении и макете для текстовых строк.

```csharp
[Flags]
public enum StringFormatFlags
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| DirectionRightToLeft | `1` | Текст отображается справа налево. |
| DirectionVertical | `2` | Текст выровнен по вертикали. |
| FitBlackBox | `4` | Части символов могут выступать за прямоугольник макета строки. По умолчанию расположение символов изменено, чтобы избежать нависания. |
| DisplayFormatControl | `20` | Управляющие символы, такие как метка слева направо, отображаются в выходных данных с репрезентативным глифом. |
| NoFontFallback | `400` | Откат к альтернативным шрифтам для символов, не поддерживаемых в запрошенном шрифте, отключен. Любые отсутствующие символы отображаются с отсутствующим глифом в шрифтах, обычно в виде открытого квадрата. |
| MeasureTrailingSpaces | `800` | Включает конечный пробел в конце каждой строки. По умолчанию граничный прямоугольник, возвращаемый методом MeasureString, исключает пробел в конце каждой строки. Установите этот флаг, чтобы включить это пространство в измерение. |
| NoWrap | `1000` | Перенос текста между строк при форматировании внутри прямоугольника отключен. Этот флаг подразумевается, когда вместо прямоугольника передается точка или когда указанный прямоугольник имеет нулевую длину линии. |
| LineLimit | `2000` | В прямоугольнике форматирования размещаются только целые строки. По умолчанию макет продолжается до конца текста или до тех пор, пока не перестанут отображаться строки в результате отсечения, в зависимости от того, что наступит раньше. целое кратное высоты строки. Чтобы убедиться, что видны только целые строки, укажите это значение и убедитесь, что длина прямоугольника форматирования не меньше высоты одной строки. |
| NoClip | `4000` | Допускается отображение выступающих частей глифов и развернутого текста, выходящего за пределы прямоугольника форматирования. По умолчанию все части текста и глифов, выходящие за пределы прямоугольника форматирования, обрезаются. |
