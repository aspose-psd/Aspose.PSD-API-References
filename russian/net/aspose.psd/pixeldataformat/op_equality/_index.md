## PixelDataFormat Equality operator

Возвращает результат равенства для двух[`PixelDataFormat`](../) классы.

```csharp
public static bool operator ==(PixelDataFormat pixelFormat1, PixelDataFormat pixelFormat2)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| pixelFormat1 | PixelDataFormat | Первый[`PixelDataFormat`](../) сравнивать. |
| pixelFormat2 | PixelDataFormat | Второй[`PixelDataFormat`](../) сравнивать. |

### Возвращаемое значение

Верно, если оба*pixelFormat1* и*pixelFormat2* содержат одинаковые данные или оба параметра имеют значение null.
