## PsdImage.VerticalResolution property

Получает или задает разрешение по вертикали в пикселях на дюйм этого[`PsdImage`](../) .

```csharp
public override double VerticalResolution { get; set; }
```

### Стоимость имущества

Вертикальное разрешение.

### Исключения

| исключение | условие |
| --- | --- |
| [PsdImageException](../../../aspose.psd.coreexceptions.imageformats/psdimageexception/) | Ресурс ResolutionInfo не найден и не может установить правильное разрешение |

### Примечания

Значение по умолчанию для PSD равно 72, поэтому, если[`ResolutionInfoResource`](../../../aspose.psd.fileformats.psd.resources/resolutioninforesource/) не найдено, возвращается это значение.
