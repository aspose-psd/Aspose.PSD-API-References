## GifOptions.MaxDiff property

Получает или задает максимально допустимую разницу в пикселях. Если больше нуля, будет использоваться сжатие с потерями. Рекомендуемое значение для оптимального сжатия с потерями — 80. 30 — очень легкое сжатие, 200 — тяжелое. очень высокие уровни потерь не дадут такого большого выигрыша. Диапазон допустимых значений: [0, 1000].

```csharp
public int MaxDiff { get; set; }
```
