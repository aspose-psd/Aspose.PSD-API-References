## GifOptions.PixelAspectRatio property

Получает или задает соотношение сторон GIF в пикселях.

```csharp
public byte PixelAspectRatio { get; set; }
```

### Стоимость имущества

Соотношение сторон пикселя GIF.

### Примечания

Соотношение сторон пикселя — коэффициент, используемый для вычисления приблизительного соотношения сторон пикселя в исходном изображении. Если значение поля не равно 0, это приближение отношения сторон вычисляется по формуле: Соотношение сторон = (Соотношение сторон пикселя + 15) / 64 Соотношение сторон пикселя определяется как частное от пикселя. s ширина по высоте. Диапазон значений в этом поле позволяет указать от самого широкого пикселя 4:1 до самого высокого пикселя 1:4 с шагом 1/64. Значения: 0 - Информация о соотношении сторон не предоставляется. 1..255 - Значение, используемое в расчете.
