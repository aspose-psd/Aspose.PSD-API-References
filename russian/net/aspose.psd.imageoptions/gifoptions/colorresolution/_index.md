## GifOptions.ColorResolution property

Получает или задает цветовое разрешение GIF.

```csharp
public byte ColorResolution { get; set; }
```

### Стоимость имущества

Цветовое разрешение.

### Примечания

Цветовое разрешение - Количество битов на первичный доступный цвет к оригиналу изображение, минус 1. Это значение представляет размер всей палитры, из которой были выбраны цвета в графике , а не количество цветов, фактически используемых в графике. Например, если значение в этом поле равно 3, то в палитре исходного изображения было доступно 4 бита на первичный цвет для создания изображение. Это значение должно указывать на богатство исходной палитры, даже если не каждый цвет из всей палитры доступен на исходной машине.
