Пространство имен содержит классы, подходящие для экспорта, сохранения или создания файлов различных форматов.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [BmpOptions](./bmpoptions/) | Опции создания формата файла bmp. |
| [CmxRasterizationOptions](./cmxrasterizationoptions/) | параметры экспортера CMX. |
| [GifOptions](./gifoptions/) | Параметры создания формата файла gif. |
| [GraphicsOptions](./graphicsoptions/) | Представляет параметры графики для встроенного растрового изображения. |
| [Jpeg2000Options](./jpeg2000options/) | Параметры формата файла Jpeg2000. |
| [JpegOptions](./jpegoptions/) | Параметры создания файла формата jpeg. |
| [MultiPageOptions](./multipageoptions/) | Базовый класс для нескольких поддерживаемых форматов страниц |
| [PdfOptions](./pdfoptions/) | Параметры PDF. |
| [PngOptions](./pngoptions/) | Параметры создания файла формата png. |
| [PsdOptions](./psdoptions/) | Параметры создания файла формата psd. |
| [RdOptimizerSettings](./rdoptimizersettings/) | Настройки оптимизатора удаленных рабочих столов class |
| [RenderResult](./renderresult/) | Представляет информацию с результатами рендеринга |
| [TiffOptions](./tiffoptions/) | Параметры формата файла tiff. Обратите внимание, что теги ширины и высоты будут перезаписаны при создании изображения параметрами ширины и высоты, поэтому нет необходимости указывать их напрямую. Обратите внимание, что многие параметры возвращают значение по умолчанию, но это не означает, что этот параметр задается явно как значение тега. Чтобы проверить наличие тега, используйте свойство Tags или соответствующий метод IsTagPresent. |
| [TiffOptionsUtils](./tiffoptionsutils/) | Служебный класс опций формата файла tiff. |
| [VectorRasterizationOptions](./vectorrasterizationoptions/) | Параметры векторной растеризации. |
