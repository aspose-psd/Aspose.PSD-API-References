## SetSigmaBellShape(float) {#setsigmabellshape}

Создает градиентную кисть, которая меняет цвет, начиная с центра пути наружу и заканчивая его границей. Переход от одного цвета к другому основан на колоколообразной кривой.

```csharp
public void SetSigmaBellShape(float focus)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| focus | Single | Значение от 0 до 1, указывающее, где вдоль любого радиуса от центра пути к границе пути центральный цвет будет иметь максимальную интенсивность. Значение 1 (по умолчанию) помещает самую высокую интенсивность в центр пути. |

## SetSigmaBellShape(float, float) {#setsigmabellshape_1}

Создает градиентную кисть, которая меняет цвет, начиная с центра пути наружу и заканчивая его границей. Переход от одного цвета к другому основан на колоколообразной кривой.

```csharp
public void SetSigmaBellShape(float focus, float scale)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| focus | Single | Значение от 0 до 1, указывающее, где вдоль любого радиуса от центра пути к границе пути центральный цвет будет иметь максимальную интенсивность. Значение 1 (по умолчанию) помещает самую высокую интенсивность в центр пути. |
| scale | Single | Значение от 0 до 1, указывающее максимальную интенсивность центрального цвета, который смешивается с граничным цветом. Значение 1 вызывает максимально возможную интенсивность центрального цвета и является значением по умолчанию. |
