## TiffDataType.CompareTo method

Сравнивает текущий экземпляр с другим объектом того же типа и возвращает целое число, указывающее, предшествует ли текущий экземпляр, следует ли за ним или находится в той же позиции в порядке сортировки как другой объект.

```csharp
public int CompareTo(object obj)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | Object | Объект для сравнения с этим экземпляром. |

### Возвращаемое значение

32-битное целое число со знаком, указывающее относительный порядок сравниваемых объектов. Возвращаемое значение имеет следующие значения: Значение Значение Меньше нуля Этот экземпляр меньше*объект*. Zero Этот экземпляр равен*obj*. Больше нуля Этот экземпляр больше, чем*obj*.
