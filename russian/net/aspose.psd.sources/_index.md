Пространство имен содержит различные источники потоков, которые подходят для потока входных или выходных данных.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [FileCreateSource](./filecreatesource/) | Представляет источник файла для создания. |
| [FileOpenSource](./fileopensource/) | Представляет источник файла для открытия. |
| [FileSource](./filesource/) | Представляет источник файлов, способный манипулировать файлами. |
| [StreamSource](./streamsource/) | Представляет источник потока. |
