## Jpeg2000LoadOptions.MaximumDecodingTime property

Получает или задает максимальное время декодирования в секундах (эту опцию можно использовать на очень медленных машинах с памятью, чтобы предотвратить зависание процесса на очень больших изображениях — разрешение более 5500x6500 пикселей).

```csharp
public int MaximumDecodingTime { get; set; }
```
