## PdfCoreOptions.HeadingsOutlineLevels property

Указывает, сколько уровней элементов структуры следует включить в структуру документа. 0 — без структуры, 1 — один уровень структуры и т. д. Значение по умолчанию: 0.

public int HeadingsOutlineLevels { get; set; }
