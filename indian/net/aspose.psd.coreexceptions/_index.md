## कक्षाओं

| कक्षा | विवरण |
| --- | --- |
| [CompressorException](./compressorexception/) | कंप्रेसर अपवाद। |
| [DataMissmatchError](./datamissmatcherror/) | डेटा बेमेल अपवाद वर्ग |
| [FrameworkException](./frameworkexception/) | PSD फ्रेमवर्क अपवाद। यह वर्ग सभी Aspose.PSD अपवादों के लिए एक मुख्य वर्ग है। |
| [ImageCreateException](./imagecreateexception/) | छवि अपवाद बनाती है। छवि निर्माण के दौरान होता है. |
| [ImageException](./imageexception/) | छवि अपवाद। |
| [ImageLoadException](./imageloadexception/) | छवि लोड अपवाद। छवि लोड करने के दौरान होता है. |
| [ImageSaveException](./imagesaveexception/) | इमेज सेव अपवाद। छवि बचत के दौरान होता है. |
| [IndexOutOFRangeException](./indexoutofrangeexception/) | कंप्रेसर अपवाद। |
| [LimitMemoryException](./limitmemoryexception/) | सीमा स्मृति अपवाद। तब होता है जब स्मृति उपयोग कम किया जाना चाहिए। |
| [OperationInterruptedException](./operationinterruptedexception/) | तब होता है जब कोई ऑपरेशन बाधित होता है। |
| [RdOptimizationError](./rdoptimizationerror/) | आरडी अनुकूलन त्रुटि अपवाद वर्ग |
| [StreamReadException](./streamreadexception/) | स्ट्रीम रीडिंग अपवाद। गलत ऑफसेट और बाइट काउंट अनुरोध के कारण स्ट्रीम रीडिंग विफल होने के कारण। |
| [XmpException](./xmpexception/) | एक्सएमपी की अमान्य संरचना होने पर फेंका गया अपवाद। |
