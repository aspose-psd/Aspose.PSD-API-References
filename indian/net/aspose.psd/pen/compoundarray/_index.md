## Pen.CompoundArray property

एक कंपाउंड पेन निर्दिष्ट करने वाले मानों की एक सरणी प्राप्त या सेट करता है। एक यौगिक कलम समानांतर रेखाओं और रिक्त स्थानों से बनी एक मिश्रित रेखा खींचती है।

```csharp
public float[] CompoundArray { get; set; }
```

### संपत्ति मूल्य

वास्तविक संख्याओं की एक सरणी जो यौगिक सरणी निर्दिष्ट करती है। सरणी में तत्वों को बढ़ते क्रम में होना चाहिए, 0 से कम नहीं और 1 से अधिक नहीं होना चाहिए।
