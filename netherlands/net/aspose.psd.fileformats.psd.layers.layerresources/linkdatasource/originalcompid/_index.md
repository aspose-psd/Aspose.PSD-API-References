## LinkDataSource.OriginalCompId property

Haalt de originele ID op van de momenteel geselecteerde Comp voor het onderliggende document, die -1 zal zijn als er geen is geselecteerd. Deze eigenschap haalt de originele laag Comp-selectie-ID op voor slimme objecten. [Laagcomposities in slimme objecten](https://helpx.adobe.com/photoshop/using/layer-comps.html)

public int OriginalCompId { get; }

### Eigendoms-waarde

De oorspronkelijke ID van de momenteel geselecteerde comp voor het onderliggende document, die -1 is als er geen is geselecteerd.
