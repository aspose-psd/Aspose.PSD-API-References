## CmykColorHelper class

Helpermethoden om met CMYK-kleuren te werken, gepresenteerd als een 32-bits integerwaarde met teken. Biedt dezelfde API als de[`CmykColor`](../cmykcolor/)struct. Het is lichter omdat CMYK-kleur net als Int32 wordt gepresenteerd in plaats van structuur met interne velden. Gebruik indien mogelijk liever statische methoden van deze klasse in plaats van de verouderde [`CmykColor`](../cmykcolor/) struct.

public static class CmykColorHelper
