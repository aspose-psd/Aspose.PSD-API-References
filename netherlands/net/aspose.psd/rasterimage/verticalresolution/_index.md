## RasterImage.VerticalResolution property

Hiermee wordt de verticale resolutie, in pixels per inch, hiervan opgehaald of ingesteld[`RasterImage`](../) .

```csharp
public virtual double VerticalResolution { get; set; }
```

### Eigendoms-waarde

De verticale resolutie.

### Opmerkingen

Merk op dat deze waarde standaard altijd 96 is, omdat verschillende platforms de schermresolutie niet kunnen retourneren. U kunt overwegen om de SetResolution-methode te gebruiken om beide resolutiewaarden in één gesprek bij te werken.
