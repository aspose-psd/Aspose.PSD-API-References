## TransparencySupporter.Opacity property

Hiermee wordt de dekking van het object opgehaald of ingesteld. De waarde moet tussen 0 en 1 liggen. De waarde 0 betekent dat het object volledig zichtbaar is, de waarde 1 betekent dat het object volledig ondoorzichtig is.

public float Opacity { get; set; }
