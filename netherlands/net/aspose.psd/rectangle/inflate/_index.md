## Inflate(Rectangle, int, int)

Creëert en retourneert een opgeblazen kopie van het gespecificeerde[`Rectangle`](../)structuur. De kopie wordt opgeblazen met de opgegeven hoeveelheid. Het origineel[`Rectangle`](../) structuur blijft ongewijzigd.

```csharp
public static Rectangle Inflate(Rectangle rect, int x, int y)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| rect | Rectangle | De[`Rectangle`](../) waarmee te beginnen. Deze rechthoek wordt niet gewijzigd. |
| x | Int32 | Het bedrag om dit op te blazen[`Rectangle`](../) horizontaal. |
| y | Int32 | Het bedrag om dit op te blazen[`Rectangle`](../) verticaal. |

### Winstwaarde

De opgeblazen[`Rectangle`](../).
