## Point Inequality operator

Vergelijkt er twee[`Point`](../) voorwerpen. Het resultaat geeft aan of de waarden van de[`X`](../x/) of[`Y`](../y/) eigenschappen van de twee[`Point`](../) objecten zijn ongelijk.

```csharp
public static bool operator !=(Point point1, Point point2)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| point1 | Point | Een eerste[`Point`](../) te vergelijken. |
| point2 | Point | Een seconde[`Point`](../) te vergelijken. |

### Winstwaarde

Waar als de waarden van ofwel de[`X`](../x/) eigenschappen of de[`Y`](../y/) eigenschappen van*point1* En*point2* verschillen; anders, vals.
