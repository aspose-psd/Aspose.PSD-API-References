## ColorMatrix class

Definieert een 5 x 5 matrix die de coördinaten voor de RGBA-ruimte bevat. Verschillende methodes van de[`ImageAttributes`](../imageattributes/) class beeldkleuren aanpassen met behulp van een kleurenmatrix. Deze klasse kan niet worden geërfd.
