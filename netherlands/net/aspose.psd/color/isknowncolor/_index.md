## Color.IsKnownColor property

Krijgt een waarde die aangeeft of dit[`Color`](../) structuur is een vooraf gedefinieerde kleur. Voorgedefinieerde kleuren worden vertegenwoordigd door de elementen van de[`KnownColor`](../../knowncolor/) opsomming.

```csharp
public bool IsKnownColor { get; }
```

### Eigendoms-waarde

Klopt als dit[`Color`](../) is gemaakt op basis van een vooraf gedefinieerde kleur met behulp van de[`FromName`](../fromname/) methode of de[`FromKnownColor`](../fromknowncolor/) methode; anders, vals.
