## FontSettings.GetFontsFolders method

Haalt een kopie op van de array die de lijst met mappen bevat waarin Aspose.Words zoekt naar TrueType-lettertypen.

```csharp
public static string[] GetFontsFolders()
```

### Winstwaarde

Een kopie van de huidige lettertypelocaties.

### Opmerkingen

De geretourneerde waarde is een kopie van de gegevens die Aspose.Words gebruikt. Als u de vermeldingen in de geretourneerde array wijzigt, , heeft dit geen effect op de documentweergave. Gebruik de[`SetFontsFolders`](../setfontsfolders/) methode.
