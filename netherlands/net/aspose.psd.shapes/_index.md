De naamruimte bevat verschillende vormen gecombineerd uit vormsegmenten.

## Klassen

| Klas | Beschrijving |
| --- | --- |
| [ArcShape](./arcshape/) | Vertegenwoordigt een boogvorm. |
| [BezierShape](./beziershape/) | Vertegenwoordigt een bezier-spline. |
| [CurveShape](./curveshape/) | Vertegenwoordigt een gebogen spline-vorm. |
| [EllipseShape](./ellipseshape/) | Vertegenwoordigt een ellipsvorm. |
| [PieShape](./pieshape/) | Vertegenwoordigt een taartvorm. |
| [PolygonShape](./polygonshape/) | Vertegenwoordigt een polygoonvorm. |
| [RectangleProjectedShape](./rectangleprojectedshape/) | Vertegenwoordigt een vorm die wordt geprojecteerd over een rechthoek die in een bepaalde richting is gedraaid. Gespecificeerd door vier punten die in de ruimte kunnen worden gedraaid met behoud van dezelfde lengte van de randen en 90 graden tussen aangrenzende randen. |
| [RectangleShape](./rectangleshape/) | Vertegenwoordigt een rechthoekige vorm. |
| [TextShape](./textshape/) | Vertegenwoordigt een tekstvorm. |
