## Layer.Flags property

Haalt of stelt de laagvlaggen in. bit 0 = transparantie beschermd; bit 1 = zichtbaar; bit 2 = verouderd; bit 3 = 1 voor Photoshop 5.0 en later, geeft aan of bit 4 nuttige informatie bevat; bit 4 = pixelgegevens die niet relevant zijn voor het uiterlijk van het document.

public LayerFlags Flags { get; set; }
