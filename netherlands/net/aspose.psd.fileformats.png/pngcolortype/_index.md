## PngColorType enumeration

Vertegenwoordigt het kleurtype PNG-afbeelding.

```csharp
public enum PngColorType
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| Grayscale | `0` | Vertegenwoordigt het kleurtype waarbij elke pixel een grijswaardenvoorbeeld is. |
| Truecolor | `2` | Vertegenwoordigt het kleurtype waarbij elke pixel een R, G, B triple is. |
| IndexedColor | `3` | Vertegenwoordigt het kleurtype waarbij elke pixel een paletindex is; er zal een PLTE-blok verschijnen. |
| GrayscaleWithAlpha | `4` | Vertegenwoordigt het kleurtype waarbij elke pixel een grijswaardenvoorbeeld is, gevolgd door een alfavoorbeeld. |
| TruecolorWithAlpha | `6` | Vertegenwoordigt het kleurtype waarbij elke pixel een R, G, B triple is, gevolgd door een alpha-sample. |
