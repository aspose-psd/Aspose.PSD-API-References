Het png-filtertype.

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| None | `0` | Het null-filter betekent geen filtering voor rijen met afbeeldingsgegevens. |
| Sub | `1` | Het subfilter betekent dat subtractief filteren wordt toegepast op beeldgegevens. |
| Up | `2` | Het omhoogfilter betekent dat een rij-voor-rij aftrekfilter wordt toegepast. |
| Avg | `3` | Het gemiddelde filter betekent dat het gemiddelde filter wordt toegepast op afbeeldingsgegevens. |
| Paeth | `4` | Het paeth-voorspellingsfilter. |
| Adaptive | `5` | Adaptief filteren, betekent dat het opslagproces het meest geschikte filter kiest voor elke gegevensrij. Beste compressie, langzaamste uitvoeringstijd. |
