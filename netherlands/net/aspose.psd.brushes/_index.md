De naamruimte biedt hulpklassen en methoden om met verschillende penseeltypen te werken.

## Klassen

| Klas | Beschrijving |
| --- | --- |
| [HatchBrush](./hatchbrush/) | Definieert een rechthoekig penseel met een arceerstijl, een voorgrondkleur en een achtergrondkleur. Deze klasse kan niet worden geërfd. |
| [LinearGradientBrush](./lineargradientbrush/) | Kapselt een[`Brush`](../aspose.psd/brush/) met een lineair verloop. Deze klasse kan niet worden geërfd. |
| [LinearGradientBrushBase](./lineargradientbrushbase/) | Vertegenwoordigt een[`Brush`](../aspose.psd/brush/) met gradiëntmogelijkheden en geschikte eigenschappen. |
| [LinearMulticolorGradientBrush](./linearmulticolorgradientbrush/) | Vertegenwoordigt een[`Brush`](../aspose.psd/brush/) met lineair verloop gedefinieerd door meerdere kleuren en geschikte posities. Deze klasse kan niet worden geërfd. |
| [PathGradientBrush](./pathgradientbrush/) | Kapselt een[`Brush`](../aspose.psd/brush/) object met een verloop. Deze klasse kan niet worden geërfd. |
| [PathGradientBrushBase](./pathgradientbrushbase/) | Vertegenwoordigt een[`Brush`](../aspose.psd/brush/) met basispadverloopfunctionaliteit. |
| [PathMulticolorGradientBrush](./pathmulticolorgradientbrush/) | Kapselt een[`Brush`](../aspose.psd/brush/) object met een verloop. Deze klasse kan niet worden geërfd. |
| [SolidBrush](./solidbrush/) | Stevig penseel is bedoeld om continu met een specifieke kleur te tekenen. Deze klasse kan niet worden geërfd. |
| [TextureBrush](./texturebrush/) | Elke eigenschap van de[`TextureBrush`](../aspose.psd.brushes/texturebrush/) klas is een[`Brush`](../aspose.psd/brush/) object dat een afbeelding gebruikt om de binnenkant van een vorm te vullen. Deze klasse kan niet worden geërfd. |
| [TransformBrush](./transformbrush/) | EEN[`Brush`](../aspose.psd/brush/) met transformatiemogelijkheden. |
