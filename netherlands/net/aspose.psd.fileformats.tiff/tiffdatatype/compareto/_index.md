## TiffDataType.CompareTo method

Vergelijkt de huidige instantie met een ander object van hetzelfde type en retourneert een geheel getal dat aangeeft of de huidige instantie voorafgaat aan, volgt op of voorkomt op dezelfde positie in de sorteervolgorde als het andere object.

```csharp
public int CompareTo(object obj)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| obj | Object | Een object om te vergelijken met deze instantie. |

### Winstwaarde

Een 32-bits geheel getal met teken dat de relatieve volgorde aangeeft van de objecten die worden vergeleken. De geretourneerde waarde heeft de volgende betekenissen: Value Betekenis Kleiner dan nul Deze instantie is kleiner dan*obj* . Zero Deze instantie is gelijk aan*obj* . Groter dan nul Deze instantie is groter dan*obj* .
