## PdfCoreOptions.ExpandedOutlineLevels property

Geeft aan hoeveel niveaus in het documentoverzicht uitgevouwen moeten worden weergegeven wanneer het PDF-bestand wordt bekeken. 0 - het documentoverzicht wordt niet uitgevouwen. 1 - items op het eerste niveau in het document worden uitgevouwen enzovoort. Standaard is 0.

public int ExpandedOutlineLevels { get; set; }
