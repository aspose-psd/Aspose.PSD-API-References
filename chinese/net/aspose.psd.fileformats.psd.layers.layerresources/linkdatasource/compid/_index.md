## LinkDataSource.CompId property

获取或设置当前为子文档选择的 comp 的 ID，如果未选择，则为 -1。 Comp 是设计人员可以创建的页面布局的组合。使用图层复合，您可以在单个 Adobe® Photoshop® 文件中创建、管理和查看布局的多个版本 。图层复合是图层面板状态的快照。 Layer comps 保存三种类型的图层选项，但 此属性获取智能对象的 Layer Comp 选择标识符。 [智能对象中的图层复合](https://helpx.adobe.com/photoshop/using/layer-comps.html)

```csharp
public int CompId { get; set; }
```

### 适当的价值

当前为子文档选择的 comp 的 ID，如果没有选择则为 -1。
