## LinkDataSource.OriginalCompId property

获取当前为子文档选择的 Comp 的原始 ID，如果没有选择则为 -1。 该属性获取 Smart Objects 的原始图层 Comp 选择标识符。 [智能对象中的图层复合](https://helpx.adobe.com/photoshop/using/layer-comps.html)

```csharp
public int OriginalCompId { get; }
```

### 适当的价值

当前为子文档选择的comp的原始ID，如果没有选择则为-1。
