## LinkDataSource.AssetLockedState property

获取或设置一个值，指示 PSD 资产是否已锁定。 Adobe® Photoshop® СС Libraries 资产的资产锁定状态。

```csharp
public bool AssetLockedState { get; set; }
```

### 适当的价值

`真的`如果 Adobe® Photoshop® 资产被锁定；否则，`错误的` .
