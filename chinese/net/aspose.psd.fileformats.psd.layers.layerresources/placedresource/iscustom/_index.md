## PlacedResource.IsCustom property

获取或设置一个值，该值指示此实例扭曲样式是否为自定义。 如果为真，则它包含网格点。如果设置为 false，它会擦除网格点。

```csharp
public bool IsCustom { get; set; }
```

### 适当的价值

`真的`如果放置的图层具有自定义样式；否则，`错误的` .
