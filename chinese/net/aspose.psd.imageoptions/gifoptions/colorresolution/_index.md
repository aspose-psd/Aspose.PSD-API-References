## GifOptions.ColorResolution property

获取或设置 GIF 颜色分辨率。

```csharp
public byte ColorResolution { get; set; }
```

### 适当的价值

颜色分辨率。

### 评论

颜色分辨率 - 对原始图像 可用的每种原色的位数减 1。此值表示大小 选择图形中颜色的整个调色板 ，而不是图形中实际使用的颜色数。 例如，如果此字段中的值为 3，则 原始图像的调色板有 4 位每种原色可用于创建 图像。应设置此值以指示 原始调色板的丰富程度，即使并非整个 调色板中的每种颜色都在源计算机上可用。
