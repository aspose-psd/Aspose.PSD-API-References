## GifOptions.MaxDiff property

获取或设置允许的最大像素差。如果大于零，将使用有损压缩。 最佳有损压缩的推荐值为 80。30 是非常轻的压缩，200 是重的。 仅引入少量损失时效果最佳，并且由于压缩算法的限制，非常高的损失级别不会带来太多收益。 允许值的范围是 [0, 1000]。

public int MaxDiff { get; set; }
