## GifOptions.MaxDiff property

获取或设置允许的最大像素差。如果大于零，将使用有损压缩。 最佳有损压缩的推荐值为 80。30 是非常轻的压缩，200 是重的。 当只引入很少的损失时效果最好，并且由于压缩算法的限制非常高的损失水平不会带来太大的收益。 允许值的范围是[0, 1000]。

public int MaxDiff { get; set; }
