## GifOptions.MaxDiff property

获取或设置最大允许像素差。如果大于零，将使用有损压缩。 最佳有损压缩的推荐值为 80。30 是非常轻的压缩，200 是重压缩。 当只引入很少的损失时效果最好，并且由于压缩算法的限制非常高的损失水平不会带来那么多的收益。 允许值的范围是 [0, 1000].

```csharp
public int MaxDiff { get; set; }
```
