## PsdOptions.RefreshImagePreviewData property

获取或设置一个值，该值指示是否 [刷新图像预览数据] - 用于最大化与其他 PSD 图像查看器的兼容性的选项。 请注意，Compact Framework 平台不支持绘制到最终布局的文本层

```csharp
public bool RefreshImagePreviewData { get; set; }
```

### 适当的价值

`真的` if [刷新图像预览数据];否则，`错误的` .
