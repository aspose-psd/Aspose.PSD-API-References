## TiffOptions.BitsPerSample property

获取或设置每个样本的位数。

```csharp
public ushort[] BitsPerSample { get; set; }
```

### 适当的价值

每个样本值的位数。

### 评论

设置此值时请记住，它还会将 SamplesPerPixel 值设置为数组长度。这 2 个属性紧密耦合，因此只能一起设置。
