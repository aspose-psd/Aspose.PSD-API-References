## PsdImage.HasTransparencyData property

获取或设置一个值，该值指示在指定图层数据时第一个 alpha 通道是否包含合并结果的透明度数据。

```csharp
public bool HasTransparencyData { get; set; }
```

### 适当的价值

`真的`如果第一个 alpha 通道包含指定图层数据时合并结果的透明度数据；否则，`错误的` .
