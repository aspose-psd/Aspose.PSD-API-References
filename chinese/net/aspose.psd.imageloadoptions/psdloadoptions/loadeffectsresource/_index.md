## PsdLoadOptions.LoadEffectsResource property

获取或设置一个值，该值指示是否[加载效果资源]（默认情况下不加载资源）。设置此选项时，只有支持的效果才会渲染到最终合并图像。

```csharp
public bool LoadEffectsResource { get; set; }
```

### 适当的价值

`真的`如果[加载效果资源]；否则，`错误的` .
