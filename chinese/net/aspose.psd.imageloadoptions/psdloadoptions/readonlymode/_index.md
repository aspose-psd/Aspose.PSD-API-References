## PsdLoadOptions.ReadOnlyMode property

获取或设置一个值，指示是否[使用只读模式]。这是只读模式，支持与 Adobe Photoshop 相同的兼容性。 设置此选项后，应用于图层的所有更改都不会保存到最终图像。所有数据均来自 ImageData 部分，因此它与 Photoshop 相同。 默认情况下，所有加载的图像都与 Adobe Photoshop 兼容。

```csharp
public bool ReadOnlyMode { get; set; }
```

### 适当的价值

` true` if [使用photoshop兼容模式]；否则，` false` 。
