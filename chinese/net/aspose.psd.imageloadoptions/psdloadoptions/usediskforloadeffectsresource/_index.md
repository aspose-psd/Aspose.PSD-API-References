## PsdLoadOptions.UseDiskForLoadEffectsResource property

获取或设置一个值，指示是否[使用磁盘加载效果资源]（默认情况下使用磁盘加载效果资源，但如果将此值设置为 false，则可以使用内存）。

```csharp
public bool UseDiskForLoadEffectsResource { get; set; }
```

### 适当的价值

`真的`如果[使用磁盘加载效果资源]；否则，`错误的` .
