## Rectangle Inequality operator

测试是否两个[`Rectangle`](../)结构的位置或大小不同。

```csharp
public static bool operator !=(Rectangle left, Rectangle right)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| left | Rectangle | 这[`Rectangle`](../)不等式运算符左侧的结构。 |
| right | Rectangle | 这[`Rectangle`](../)不等式运算符右侧的结构。 |

### 返回值

如果有任何一个，则此运算符返回 true[`X`](../x/) ,[`Y`](../y/) ,[`Width`](../width/)或者[`Height`](../height/)两者的属性[`Rectangle`](../)结构不平等；否则为假。
