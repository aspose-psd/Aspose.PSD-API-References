## Brush.Opacity property

获取或设置画笔不透明度。该值应介于 0 和 1 之间。值为 0 表示画笔完全可见，值为 1 表示画笔完全不透明。

```csharp
public float Opacity { get; set; }
```

### 适当的价值

画笔不透明度值。
