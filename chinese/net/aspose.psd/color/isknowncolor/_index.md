## Color.IsKnownColor property

获取一个值，指示是否这[`Color`](../)结构是预定义的颜色。预定义的颜色由[`KnownColor`](../../knowncolor/)枚举.

```csharp
public bool IsKnownColor { get; }
```

### 适当的价值

如果这个是真的[`Color`](../)是通过使用[`FromName`](../fromname/)方法或[`FromKnownColor`](../fromknowncolor/)方法;否则，假的。
