## Color Inequality operator

测试是否指定了两个[`Color`](../)结构不同.

```csharp
public static bool operator !=(Color left, Color right)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| left | Color | 这[`Color`](../)即不等式运算符的左侧。 |
| right | Color | 这[`Color`](../)那是在不等式运算符的右边。 |

### 返回值

真如果两个[`Color`](../)结构不同；否则，假的。
