## Color Equality operator


| 范围 | 类型 | 描述 |
| --- | --- | --- |
| left | Color | 这[`Color`](../)那是在相等运算符的左边。 |
| right | Color | 这[`Color`](../)那是在相等运算符的右边。 |

### 返回值
