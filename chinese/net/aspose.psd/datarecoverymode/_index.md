## DataRecoveryMode enumeration

数据恢复模式。

```csharp
public enum DataRecoveryMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| None | `0` | 不暗示数据恢复。每当文件格式有一些损坏的数据时，就会抛出相应的异常。 |
| ConsistentRecover | `1` | 只要损坏不破坏文件格式并允许正确的进一步处理，一致恢复模式就会尝试恢复所有数据。 |
| MaximalRecover | `2` | 最大恢复模式恢复所有数据，即使文件格式结构已损坏，进一步处理可能会产生无人看管的效果。 |
