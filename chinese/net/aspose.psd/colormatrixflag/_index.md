## ColorMatrixFlag enumeration

指定将受颜色和灰度调整设置影响的图像和颜色类型[`ImageAttributes`](../imageattributes/) .

```csharp
public enum ColorMatrixFlag
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Default | `0` | 所有颜色值，包括灰色阴影，都由相同的颜色调整矩阵调整。 |
| SkipGrays | `1` | 调整了所有颜色，但未调整灰色阴影。灰色阴影是其红色、绿色和蓝色分量具有相同值的任何颜色。 |
| AltGrays | `2` | 仅调整灰色阴影。 |
