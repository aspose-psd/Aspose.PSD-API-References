## Size Inequality operator

测试是否两个[`Size`](../)结构不同.

```csharp
public static bool operator !=(Size size1, Size size2)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| size1 | Size | 这[`Size`](../)不等式运算符左边的结构。 |
| size2 | Size | 这[`Size`](../)不等式运算符右侧的结构。 |

### 返回值

真如果*size1*和*size2*宽度或高度不同；假的如果*size1*和*size2*是平等的。
