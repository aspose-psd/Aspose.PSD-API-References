## Matrix.TypeQuadrantRotation field

此标志位表示此对象 定义的变换执行象限旋转 90 度的某个倍数 in 除了其他标志位指示的转换。 旋转将矢量的角度改变相同的量 而不管原始方向如何向量的长度并且没有 改变向量的长度。 这个标志位与TypeGeneralRotation标志互斥。

```csharp
public const int TypeQuadrantRotation;
```
