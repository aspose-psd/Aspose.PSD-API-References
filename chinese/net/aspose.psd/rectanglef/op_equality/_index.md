## RectangleF Equality operator

测试是否两个[`RectangleF`](../)结构具有相同的位置和大小。

```csharp
public static bool operator ==(RectangleF left, RectangleF right)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| left | RectangleF | 这[`RectangleF`](../)等于运算符左侧的结构。 |
| right | RectangleF | 这[`RectangleF`](../)等于运算符右侧的结构。 |

### 返回值
