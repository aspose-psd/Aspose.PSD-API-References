## Figure.IsClosed property

获取或设置一个值，该值指示该图形是否闭合。只有在 where 第一个和最后一个图形的形状是连续形状的情况下，闭合图形才会有所不同。在这种情况下，第一个形状的第一个点将被 与最后一个形状的最后一个点连接起来。

```csharp
public bool IsClosed { get; set; }
```

### 适当的价值

`真的`如果这个数字是封闭的；否则，`错误的`.
