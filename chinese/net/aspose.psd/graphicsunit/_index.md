## GraphicsUnit enumeration

指定给定数据的度量单位。

```csharp
public enum GraphicsUnit
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| World | `0` | 指定世界坐标系单位为度量单位。 |
| Display | `1` | 指定显示设备的测量单位。视频显示器通常为像素，打印机为 1/100 英寸。 |
| Pixel | `2` | 将设备像素指定为度量单位。 |
| Point | `3` | 指定打印机的点（1/72 英寸）作为度量单位。 |
| Inch | `4` | 将英寸指定为度量单位。 |
| Document | `5` | 指定文档单位（1/300 英寸）为度量单位。 |
| Millimeter | `6` | 指定毫米为度量单位。 |
