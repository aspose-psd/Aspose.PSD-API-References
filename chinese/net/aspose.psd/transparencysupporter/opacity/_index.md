## TransparencySupporter.Opacity property

获取或设置对象的不透明度。该值应介于 0 和 1 之间。值为 0 表示对象完全可见，值为 1 表示对象完全不透明。

```csharp
public float Opacity { get; set; }
```

### 适当的价值

不透明度值。
