## CacheType enumeration

指定要使用的缓存类型。

```csharp
public enum CacheType
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Auto | `0` | 指定自动缓存类型。自动缓存方案将尝试先在内存中缓存，然后在磁盘上缓存。 |
| CacheOnDiskOnly | `1` | 仅磁盘缓存。 |
| CacheInMemoryOnly | `2` | 仅内存缓存。 |
