## DrawArc(Pen, float, float, float, float, float, float) {#drawarc_3}

يرسم قوسًا يمثل جزءًا من القطع الناقص المحدد بواسطة زوج من الإحداثيات والعرض والارتفاع.

```csharp
public void DrawArc(Pen pen, float x, float y, float width, float height, float startAngle, 
    float sweepAngle)
```

| معامل | يكتب | وصف |
| --- | --- | --- |
| pen | Pen | [`Pen`](../../pen/) التي تحدد لون القوس وعرضه ونمطه. |
| x | Single | إحداثي x للركن الأيسر العلوي للمستطيل الذي يحدد القطع الناقص. |
| y | Single | إحداثيات y للركن الأيسر العلوي للمستطيل الذي يحدد القطع الناقص. |
| width | Single | عرض المستطيل الذي يحدد القطع الناقص. |
| height | Single | ارتفاع المستطيل الذي يحدد القطع الناقص. |
| startAngle | Single | تُقاس الزاوية بالدرجات في اتجاه عقارب الساعة من المحور x إلى نقطة بداية القوس. |
| sweepAngle | Single | قياس الزاوية بالدرجات في اتجاه عقارب الساعة من*startAngle* المعلمة لنقطة نهاية القوس. |
