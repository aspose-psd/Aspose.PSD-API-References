## StringAlignment enumeration

Gibt die Ausrichtung einer Textzeichenfolge relativ zu ihrem Layoutrechteck an.

```csharp
public enum StringAlignment
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Near | `0` | Gibt an, dass der Text in der Nähe des Layouts ausgerichtet wird. Bei einem Layout von links nach rechts ist die nahe Position links. Bei einem Layout von rechts nach links ist die nahe Position rechts. |
