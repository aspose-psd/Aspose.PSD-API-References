## Inflate(Rectangle, int, int)

Erstellt und gibt eine vergrößerte Kopie der angegebenen zurück[`Rectangle`](../)Struktur. Die Kopie wird um den angegebenen Betrag aufgeblasen. Das Original[`Rectangle`](../) Struktur bleibt unverändert.

```csharp
public static Rectangle Inflate(Rectangle rect, int x, int y)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| rect | Rectangle | Der[`Rectangle`](../) womit anfangen. Dieses Rechteck wird nicht modifiziert. |
| x | Int32 | Der Betrag, um dies aufzublasen[`Rectangle`](../) horizontal. |
| y | Int32 | Der Betrag, um dies aufzublasen[`Rectangle`](../) vertikal. |

### Rückgabewert

Das Aufgeblasene[`Rectangle`](../).
