## IOrderedShape.IsClosed property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob die geordnete Form geschlossen ist. Bei der Verarbeitung geschlossen geordneter Formen haben die Start- und Endpunkte keine Bedeutung.

```csharp
public bool IsClosed { get; set; }
```

### Eigentumswert

`WAHR` wenn diese geordnete Form geschlossen ist; ansonsten,`FALSCH`.
