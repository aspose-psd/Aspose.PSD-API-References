## TransparencySupporter.Opacity property

Ruft die Deckkraft des Objekts ab oder legt sie fest. Der Wert sollte zwischen 0 und 1 liegen. Der Wert 0 bedeutet, dass das Objekt vollständig sichtbar ist, der Wert 1 bedeutet, dass das Objekt vollständig undurchsichtig ist.

```csharp
public float Opacity { get; set; }
```
