## SplitStreamContainer.Seek method

Legt die Position innerhalb des aktuellen Streams fest.

```csharp
public override long Seek(long offset, SeekOrigin origin)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| offset | Int64 | Ein Byte-Offset relativ zum*origin* Parameter. Dieser Wert stellt den Offset von der Startposition des Streams dar, die im StreamContainer-Konstruktor übergeben wird. |
| origin | SeekOrigin | Ein Wert vom TypSeekOrigin zeigt den Referenzpunkt an, der verwendet wird, um die neue Position zu erhalten. |

### Rückgabewert

Die neue Position innerhalb des aktuellen Streams.
