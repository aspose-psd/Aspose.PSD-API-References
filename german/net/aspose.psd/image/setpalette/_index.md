## Image.SetPalette method

Legt die Bildpalette fest.

```csharp
public abstract void SetPalette(IColorPalette palette, bool updateColors)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| palette | IColorPalette | Die einzustellende Palette. |
| updateColors | Boolean | wenn eingestellt auf`WAHR` Farben werden entsprechend der neuen Palette aktualisiert; ansonsten bleiben die Farbindizes unverändert. Beachten Sie, dass unveränderte Indizes das Bild beim Laden zum Absturz bringen können, wenn einige Indizes keine entsprechenden Paletteneinträge haben. |
