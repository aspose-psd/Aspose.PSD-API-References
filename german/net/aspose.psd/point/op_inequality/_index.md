## Point Inequality operator

Vergleicht zwei[`Point`](../) Objekte. Das Ergebnis gibt an, ob die Werte der[`X`](../x/) oder[`Y`](../y/) Eigenschaften der beiden[`Point`](../) Objekte sind ungleich.

```csharp
public static bool operator !=(Point point1, Point point2)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| point1 | Point | Eine Premiere[`Point`](../) vergleichen. |
| point2 | Point | Eine Sekunde[`Point`](../) vergleichen. |

### Rückgabewert

True, wenn die Werte von entweder der[`X`](../x/) Eigenschaften oder die[`Y`](../y/) Eigentum von*point1* Und*point2* abweichen; andernfalls falsch.
