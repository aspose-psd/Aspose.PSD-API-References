## Inflate(RectangleF, float, float)

Erstellt und gibt eine vergrößerte Kopie der angegebenen zurück[`RectangleF`](../)Struktur. Die Kopie wird um den angegebenen Betrag aufgeblasen. Das ursprüngliche Rechteck bleibt unverändert.

```csharp
public static RectangleF Inflate(RectangleF rect, float x, float y)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| rect | RectangleF | Der[`RectangleF`](../) kopiert werden. Dieses Rechteck wird nicht modifiziert. |
| x | Single | Der Betrag, um den die Kopie des Rechtecks horizontal aufgeblasen wird. |
| y | Single | Der Betrag, um den die Kopie des Rechtecks vertikal aufgebläht wird. |
