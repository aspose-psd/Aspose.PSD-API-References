## Size Inequality operator

Testet ob zwei[`Size`](../) Strukturen sind unterschiedlich.

```csharp
public static bool operator !=(Size size1, Size size2)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| size1 | Size | Der[`Size`](../) Struktur links vom Ungleichheitsoperator. |
| size2 | Size | Der[`Size`](../) Struktur rechts vom Ungleichheitsoperator. |

### Rückgabewert

Wahr, wenn*size1* Und*size2* unterscheiden sich entweder in Breite oder Höhe; falsch wenn*size1* Und*size2* sind gleich.
