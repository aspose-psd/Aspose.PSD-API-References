## LineJoin enumeration

Gibt an, wie aufeinanderfolgende Linien- oder Kurvensegmente in einer Figur (Teilpfad) verbunden werden, die in a enthalten ist[`GraphicsPath`](../graphicspath/) Objekt.

```csharp
public enum LineJoin
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Miter | `0` | Gibt eine Gehrungsverbindung an. Dies erzeugt eine scharfe Ecke oder eine abgeschnittene Ecke, je nachdem, ob die Länge der Gehrung die Gehrungsgrenze überschreitet. |
| Bevel | `1` | Gibt eine abgeschrägte Verbindung an. Dadurch entsteht eine diagonale Ecke. |
| Round | `2` | Gibt einen kreisförmigen Join an. Dadurch entsteht ein glatter, kreisförmiger Bogen zwischen den Linien. |
| MiterClipped | `3` | Gibt eine Gehrungsverbindung an. Dies erzeugt eine scharfe Ecke oder eine abgeschrägte Ecke, je nachdem, ob die Gehrungslänge die Gehrungsgrenze überschreitet. |
