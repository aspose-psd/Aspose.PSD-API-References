## IPlacedLayerResource.IsCustom property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob dieser Instanz-Warp-Stil benutzerdefiniert ist. Wenn wahr, enthält er Mesh-Punkte. Wenn auf „false“ gesetzt, werden Mesh-Punkte gelöscht.

```csharp
public bool IsCustom { get; set; }
```

### Eigentumswert

`WAHR` wenn die platzierte oder Smart-Objekt-Layer-Ressource einen benutzerdefinierten Stil hat; ansonsten,`FALSCH` .
