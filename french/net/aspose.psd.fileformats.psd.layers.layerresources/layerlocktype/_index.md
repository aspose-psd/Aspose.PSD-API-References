## LayerLockType enumeration

Options de verrouillage de calque

```csharp
[Flags]
public enum LayerLockType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Pas de verrouillage de calque |
| LockTransparentPixels | `1` | Verrouiller partiellement un calque - Limite l'édition aux parties opaques du calque. Cette option est équivalente à l'option Préserver la transparence des versions antérieures de Photoshop. |
| LockImagePixels | `2` | Verrouiller partiellement un calque - Empêche la modification des pixels du calque à l'aide des outils de peinture. |
| LockPosition | `4` | Verrouiller partiellement un calque - Empêche le déplacement des pixels du calque. |
| LockAll | `7` | Verrouiller toutes les propriétés d'un calque |
