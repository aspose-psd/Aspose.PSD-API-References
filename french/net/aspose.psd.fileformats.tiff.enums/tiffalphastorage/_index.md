## TiffAlphaStorage enumeration

Spécifie le stockage alpha pour les documents tiff.

```csharp
public enum TiffAlphaStorage : ushort
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Unspecified | `0` | L'alpha n'est pas spécifié et stocké dans le fichier tiff. |
| Associated | `1` | La valeur alpha est stockée sous forme prémultipliée. Lorsque l'alpha est restauré, il peut y avoir des effets d'arrondi et la valeur restaurée peut être différente de l'original. |
| Unassociated | `2` | La valeur alpha est stockée sous forme non associée. Cela signifie que l'alpha restauré est exactement le même que celui qui a été stocké dans le tiff. |
