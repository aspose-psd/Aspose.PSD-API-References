## ThumbnailResource.JpegOptions property

Obtient ou définit les options JPEG. Convient lorsque la ressource de vignette est enregistrée au format de fichier JPEG uniquement. Cette option n'a aucun effet lorsque le format RAW est défini.

```csharp
public JpegOptions JpegOptions { get; set; }
```

### Valeur de la propriété

Les options JPEG.
