## TransparencySupporter.Opacity property

Obtient ou définit l'opacité de l'objet. La valeur doit être comprise entre 0 et 1. La valeur 0 signifie que l'objet est entièrement visible, la valeur 1 signifie que l'objet est entièrement opaque.

```csharp
public float Opacity { get; set; }
```

### Valeur de la propriété

La valeur d'opacité.
