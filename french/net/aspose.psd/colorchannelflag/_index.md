## ColorChannelFlag enumeration

Spécifie les canaux individuels dans l'espace colorimétrique CMJN (cyan, magenta, jaune, noir). Cette énumération est utilisée par les méthodes SetOutputChannel.

```csharp
public enum ColorChannelFlag
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| ColorChannelC | `0` | Le canal de couleur cyan. |
| ColorChannelM | `1` | Le canal de couleur magenta. |
| ColorChannelY | `2` | Le canal de couleur jaune. |
| ColorChannelK | `3` | Le canal de couleur noire. |
| ColorChannelLast | `4` | Le dernier canal sélectionné doit être utilisé. |
