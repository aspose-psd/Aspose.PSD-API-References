## LineJoin enumeration

Spécifie comment joindre des segments de ligne ou de courbe consécutifs dans une figure (sous-chemin) contenue dans un[`GraphicsPath`](../graphicspath/) objet.

```csharp
public enum LineJoin
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Miter | `0` | Spécifie une jointure en onglet. Cela produit un angle vif ou un angle coupé, selon que la longueur de l'onglet dépasse la limite d'onglet. |
| Bevel | `1` | Spécifie une jointure biseautée. Cela produit un coin diagonal. |
| Round | `2` | Spécifie une jointure circulaire. Cela produit un arc de cercle lisse entre les lignes. |
| MiterClipped | `3` | Spécifie une jointure en onglet. Cela produit un coin pointu ou un coin biseauté, selon que la longueur de l'onglet dépasse la limite d'onglet. |
