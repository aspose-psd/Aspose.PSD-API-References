## LineCap enumeration

Spécifie les styles de capuchon disponibles avec lesquels un[`Pen`](../pen/) l'objet peut terminer une ligne.

```csharp
public enum LineCap
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Flat | `0` | Spécifie une limite de ligne plate. |
| Square | `1` | Spécifie une limite de ligne carrée. |
| Round | `2` | Spécifie une limite de ligne ronde. |
| Triangle | `3` | Spécifie un chapeau de ligne triangulaire. |
| NoAnchor | `16` | Ne spécifie aucune ancre. |
| SquareAnchor | `17` | Spécifie une tête de ligne d'ancrage carrée. |
| RoundAnchor | `18` | Spécifie un capuchon d'ancrage rond. |
| DiamondAnchor | `19` | Spécifie un capuchon d'ancrage en diamant. |
| ArrowAnchor | `20` | Spécifie un capuchon d'ancrage en forme de flèche. |
| AnchorMask | `240` | Spécifie une limite de ligne personnalisée. |
| Custom | `255` | Spécifie un masque utilisé pour vérifier si un capuchon de ligne est un capuchon d'ancrage. |
