## Color.IsKnownColor property

Obtient une valeur indiquant si cela[`Color`](../) la structure est une couleur prédéfinie. Les couleurs prédéfinies sont représentées par les éléments du[`KnownColor`](../../knowncolor/) énumération.

```csharp
public bool IsKnownColor { get; }
```

### Valeur de la propriété

Vrai si cela[`Color`](../) a été créé à partir d'une couleur prédéfinie en utilisant soit[`FromName`](../fromname/) méthode ou la[`FromKnownColor`](../fromknowncolor/) méthode; sinon, faux.
