## PixelDataFormat.RgbIndexed8Bpp property

Obtient le[`PixelDataFormat`](../) défini pour 8 bits indexés par couleur. Le stockage de données de pixels indexés est destiné à permettre le stockage et la récupération de données partout où la palette de couleurs est utilisée. À utiliser avec prudence, car peut nécessiter une conversion d'une palette à une autre ou de RVBA à un modèle de couleurs indexées .

```csharp
public static PixelDataFormat RgbIndexed8Bpp { get; }
```

### Valeur de la propriété

Le[`PixelDataFormat`](../) défini pour indexé 8 bits par couleur.
