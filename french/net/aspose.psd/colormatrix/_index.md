## ColorMatrix class

Définit une matrice 5 x 5 qui contient les coordonnées de l'espace RGBA. Plusieurs méthodes de la[`ImageAttributes`](../imageattributes/) ajuste les couleurs de l'image à l'aide d'une matrice de couleurs. Cette classe ne peut pas être héritée.

public sealed class ColorMatrix
