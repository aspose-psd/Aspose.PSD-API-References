## Layer.Flags property

Obtient ou définit les indicateurs de calque. bit 0 = transparence protégée ; bit 1 = visible ; bit 2 = obsolète ; bit 3 = 1 pour Photoshop 5.0 et versions ultérieures, indique si le bit 4 contient des informations utiles ; bit 4 = données de pixel sans rapport avec l'apparence du document.

```csharp
public LayerFlags Flags { get; set; }
```

### Valeur de la propriété

La couche flags.
