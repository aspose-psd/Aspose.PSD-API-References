## PsdLoadOptions.ReadOnlyMode property

Obtient ou définit une valeur indiquant si [utiliser le mode lecture seule]. Il s'agit du mode lecture seule, pris en charge pour une compatibilité identique avec Adobe Photoshop. Lorsque cette option est définie, toutes les modifications appliquées aux calques ne seront pas enregistrées dans l'image finale. Toutes les données sont utilisées à partir de la section ImageData, elles sont donc identiques à Photoshop. Par défaut, toutes les images chargées ne sont pas identiques à Adobe Photoshop compatible.

```csharp
public bool ReadOnlyMode { get; set; }
```

### Valeur de la propriété

`vrai` si [utiliser le mode de compatibilité photoshop] ; sinon,`FAUX` .
