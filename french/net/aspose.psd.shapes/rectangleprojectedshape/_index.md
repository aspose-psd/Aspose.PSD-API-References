## RectangleProjectedShape class

Représente une forme qui est projetée sur un rectangle tourné vers une orientation particulière. Spécifié par quatre points qui peuvent être pivotés dans l'espace en maintenant la même longueur d'arêtes et 90 degrés entre les arêtes adjacentes.

```csharp
public abstract class RectangleProjectedShape : Shape
```
