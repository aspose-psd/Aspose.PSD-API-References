## Point Equality operator

Membandingkan dua[`Point`](../) objek. Hasilnya menentukan apakah nilai-nilai dari[`X`](../x/) Dan[`Y`](../y/) sifat keduanya[`Point`](../) objek sama.

```csharp
public static bool operator ==(Point point1, Point point2)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| point1 | Point | Yang pertama[`Point`](../) untuk membandingkan. |
| point2 | Point | Sebentar[`Point`](../) untuk membandingkan. |

### Nilai Pengembalian

Benar jika[`X`](../x/) Dan[`Y`](../y/) nilai dari*point1* Dan*point2* adalah sama; jika tidak, salah.
