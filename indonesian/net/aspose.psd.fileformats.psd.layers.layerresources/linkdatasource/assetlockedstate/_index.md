## LinkDataSource.AssetLockedState property

Mendapat atau menyetel nilai yang menunjukkan apakah aset PSD dikunci. Status aset terkunci, untuk aset Perpustakaan Adobe® Photoshop® СС.

```csharp
public bool AssetLockedState { get; set; }
```

### Nilai properti

`BENAR` jika aset Adobe® Photoshop® dikunci; jika tidak,`PALSU` .
