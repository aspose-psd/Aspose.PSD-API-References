## TiffExpectedFormat enumeration

Format file tiff yang diharapkan.

```csharp
public enum TiffExpectedFormat
```

### Nilai

| Nama | Nilai | Keterangan |
| --- | --- | --- |
| Default | `0` | Format tiff default adalah tanpa kompresi dengan format B/W 1 bit per piksel saja. Anda juga dapat menggunakan setelan ini untuk mendapatkan opsi kosong dan melakukan inisialisasi dengan tag Anda atau setelan lainnya. |
| TiffLzwBw | `1` | Tiff memiliki kompresi LZW dan format B/W 1 bit per piksel saja. |
| TiffLzwRgb | `2` | Tiff memiliki kompresi LZW dan format warna RGB. |
| TiffLzwRgba | `3` | Tiff memiliki kompresi LZW dan RGBA dengan format warna transparansi. |
| TiffLzwCmyk | `4` | Tiff LZW cmyk |
| TiffCcittFax3 | `5` | Penyandian tiff CCITT FAX3. B/W 1 bit per piksel hanya didukung untuk skema tersebut. |
| TiffCcittFax4 | `6` | Penyandian tiff CCITT FAX4. B/W 1 bit per piksel hanya didukung untuk skema tersebut. |
| TiffDeflateBw | `7` | Tiff memiliki kompresi deflate dan format B/W 1 bit per piksel saja. |
| TiffDeflateRgb | `8` | Tiff memiliki kompresi deflate dan format warna RGB. |
| TiffDeflateRgba | `9` | Tiff memiliki kompresi deflate dan format warna RGBA. |
| TiffCcitRle | `10` | Penyandian RLE CCITT tiff. B/W 1 bit per piksel hanya didukung untuk skema tersebut. |
| TiffJpegRgb | `11` | Tiff memiliki kompresi Jpeg dan format warna RGB. |
| TiffJpegYCbCr | `12` | Tiff memiliki kompresi Jpeg dan format warna YCBCR. |
| TiffNoCompressionBw | `13` | Hanya format tiff dan B/W 1 bit per piksel yang tidak terkompresi. |
| TiffNoCompressionRgb | `14` | Format warna tiff dan RGB yang tidak terkompresi. |
| TiffNoCompressionRgba | `15` | Tiff dan RGBA tanpa kompresi dengan format warna transparansi. |
