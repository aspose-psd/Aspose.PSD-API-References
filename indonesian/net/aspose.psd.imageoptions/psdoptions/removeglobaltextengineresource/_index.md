## PsdOptions.RemoveGlobalTextEngineResource property

Mendapat atau menyetel nilai yang menunjukkan apakah - Hapus sumber daya mesin teks global - Digunakan untuk beberapa file psd berlapis teks, hanya jika file tersebut tidak dapat dibuka di Adobe Photoshop setelah diproses (kebanyakan untuk font yang tidak ada terkait lapisan teks). Setelah menggunakan opsi ini, pengguna perlu Membuat file selanjutnya yang dibuka di Photoshop: Menu "Teks" -&gt; "Proses font yang tidak ada". Setelah operasi itu, semua teks akan muncul lagi. Perhatikan, bahwa operasi ini dapat menyebabkan beberapa perubahan tata letak akhir.

```csharp
public bool RemoveGlobalTextEngineResource { get; set; }
```

### Nilai properti

`BENAR` if [hapus sumber daya mesin teks global]; jika tidak,`PALSU` .
