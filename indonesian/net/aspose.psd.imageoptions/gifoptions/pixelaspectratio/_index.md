## GifOptions.PixelAspectRatio property

Mendapatkan atau menyetel rasio aspek piksel GIF.

```csharp
public byte PixelAspectRatio { get; set; }
```

### Nilai properti

Rasio aspek piksel GIF.

### Perkataan

Rasio Aspek Piksel - Faktor yang digunakan untuk menghitung perkiraan rasio aspek piksel pada gambar asli. Jika nilai bidang bukan 0, perkiraan rasio aspek ini dihitung berdasarkan rumus: Rasio Aspek = (Rasio Aspek Piksel + 15) / 64 Rasio Aspek Piksel didefinisikan sebagai hasil bagi piksel' s lebarnya melebihi tingginya. Rentang nilai dalam bidang ini memungkinkan spesifikasi piksel terlebar 4:1 hingga piksel tertinggi 1:4 dengan kelipatan 1/64. Nilai : 0 - Tidak ada informasi rasio aspek yang diberikan. 1..255 - Nilai yang digunakan dalam perhitungan.
