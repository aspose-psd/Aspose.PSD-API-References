Ruang nama berisi pengecualian yang dilemparkan oleh salah satu komponen inti PSD.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [CompressorException](./compressorexception/) | Pengecualian kompresor. |
| [DataMissmatchError](./datamissmatcherror/) | Kelas pengecualian ketidakcocokan data |
| [FrameworkException](./frameworkexception/) | Pengecualian framework PSD. Kelas ini adalah kelas inti untuk semua pengecualian Aspose.PSD. Diperkenalkan untuk membedakan antara pengecualian yang dilemparkan oleh mesin Aspose.PSD dan semua jenis pengecualian lainnya. |
| [ImageCreateException](./imagecreateexception/) | Gambar membuat pengecualian. Terjadi selama pembuatan gambar. |
| [ImageException](./imageexception/) | Pengecualian gambar. |
| [ImageLoadException](./imageloadexception/) | Pengecualian pemuatan gambar. Terjadi selama pemuatan gambar. |
| [ImageSaveException](./imagesaveexception/) | Pengecualian penyimpanan gambar. Terjadi selama penyimpanan gambar. |
| [IndexOutOFRangeException](./indexoutofrangeexception/) | Pengecualian kompresor. |
| [LimitMemoryException](./limitmemoryexception/) | Pengecualian batas memori. Terjadi saat penggunaan memori harus dikurangi. |
| [OperationInterruptedException](./operationinterruptedexception/) | Terjadi saat operasi terganggu. |
| [RdOptimizationError](./rdoptimizationerror/) | kelas pengecualian kesalahan pengoptimalan RD |
| [StreamReadException](./streamreadexception/) | Pengecualian pembacaan aliran. Disebabkan ketika pembacaan aliran gagal karena offset yang salah dan permintaan jumlah byte. |
| [XmpException](./xmpexception/) | Pengecualian yang dilontarkan saat XMP memiliki struktur yang tidak valid. |
