Ruang nama menyediakan kelas pembantu dan metode untuk bekerja dengan berbagai jenis kuas.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [HatchBrush](./hatchbrush/) | Menentukan sikat persegi panjang dengan gaya penetasan, warna latar depan, dan warna latar belakang. Kelas ini tidak dapat diwariskan. |
| [LinearGradientBrush](./lineargradientbrush/) | Merangkum a[`Brush`](../aspose.psd/brush/) dengan gradien linier. Kelas ini tidak dapat diwariskan. |
| [LinearGradientBrushBase](./lineargradientbrushbase/) | Mewakili a[`Brush`](../aspose.psd/brush/) dengan kemampuan gradien dan properti yang sesuai. |
| [LinearMulticolorGradientBrush](./linearmulticolorgradientbrush/) | Mewakili a[`Brush`](../aspose.psd/brush/) dengan gradien linier yang ditentukan oleh berbagai warna dan posisi yang sesuai. Kelas ini tidak dapat diwariskan. |
| [PathGradientBrush](./pathgradientbrush/) | Merangkum a[`Brush`](../aspose.psd/brush/) objek dengan gradien. Kelas ini tidak dapat diwariskan. |
| [PathGradientBrushBase](./pathgradientbrushbase/) | Mewakili a[`Brush`](../aspose.psd/brush/) dengan fungsionalitas gradien jalur dasar. |
| [PathMulticolorGradientBrush](./pathmulticolorgradientbrush/) | Merangkum a[`Brush`](../aspose.psd/brush/) objek dengan gradien. Kelas ini tidak dapat diwariskan. |
| [SolidBrush](./solidbrush/) | Kuas padat dimaksudkan untuk menggambar terus menerus dengan warna tertentu. Kelas ini tidak dapat diwariskan. |
| [TextureBrush](./texturebrush/) | Setiap properti dari[`TextureBrush`](../aspose.psd.brushes/texturebrush/) kelas adalah[`Brush`](../aspose.psd/brush/) objek yang menggunakan gambar untuk mengisi interior bentuk. Kelas ini tidak dapat diwariskan. |
| [TransformBrush](./transformbrush/) | A[`Brush`](../aspose.psd/brush/) dengan kemampuan transformasi. |
