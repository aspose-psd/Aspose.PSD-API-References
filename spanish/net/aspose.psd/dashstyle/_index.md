## DashStyle enumeration

Especifica el estilo de las líneas discontinuas dibujadas con un[`Pen`](../pen/) objeto.

```csharp
public enum DashStyle
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Solid | `0` | Especifica una línea continua. |
| Dash | `1` | Especifica una línea que consta de guiones. |
| Dot | `2` | Especifica una línea que consta de puntos. |
| DashDot | `3` | Especifica una línea que consiste en un patrón repetitivo de guión-punto. |
| DashDotDot | `4` | Especifica una línea que consta de un patrón repetido de guión-punto-punto. |
| Custom | `5` | Especifica un estilo de guión personalizado definido por el usuario. |
