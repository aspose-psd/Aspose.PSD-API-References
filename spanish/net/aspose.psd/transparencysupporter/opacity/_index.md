## TransparencySupporter.Opacity property

Obtiene o establece la opacidad del objeto. El valor debe estar entre 0 y 1. El valor de 0 significa que el objeto es completamente visible, el valor de 1 significa que el objeto es completamente opaco.

```csharp
public float Opacity { get; set; }
```

### El valor de la propiedad

El valor de la opacidad.
