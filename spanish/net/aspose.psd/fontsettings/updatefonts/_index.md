## FontSettings.UpdateFonts method

Actualiza la caché de fuentes para archivos PSD que contienen capas de texto. Este método garantiza que las fuentes de la carpeta fontsFolder usando método FontSettings.SetFontsFolder(fontsFolder) o después de restablecer las fuentes usando FontSettings.Reset() se tendrán en cuenta al procesar archivos PSD. Utilice este método cada vez que FontSettings.SetFontsFolder(fontsFolder) o FontSettings.Reset() soliciten imágenes PSD. Sin llamar a este método, no hay garantía de que las fuentes se actualicen.

```csharp
public static void UpdateFonts()
```
