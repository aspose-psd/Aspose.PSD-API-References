## ColorMatrix class

Define una matriz de 5 x 5 que contiene las coordenadas para el espacio RGBA. Varios métodos de la[`ImageAttributes`](../imageattributes/) la clase ajusta los colores de la imagen usando una matriz de color. Esta clase no se puede heredar.

public sealed class ColorMatrix
