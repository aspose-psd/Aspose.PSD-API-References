## StringTrimming enumeration

Especifica cómo recortar caracteres de una cadena que no encaja completamente en una forma de diseño.

```csharp
public enum StringTrimming
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | No especifica recorte. |
| Character | `1` | Especifica que el texto se recorta al carácter más próximo. |
| Word | `2` | Especifica que el texto se recorta a la palabra más cercana. |
| EllipsisCharacter | `3` | Especifica que el texto se recorta al carácter más cercano y se inserta una elipsis al final de una línea recortada. |
| EllipsisWord | `4` | Especifica que el texto se recorta a la palabra más cercana y se inserta una elipsis al final de una línea recortada. |
| EllipsisPath | `5` | El centro se elimina de las líneas recortadas y se reemplaza por puntos suspensivos. El algoritmo conserva la mayor parte posible del último segmento de la línea delimitado por barras diagonales. |
