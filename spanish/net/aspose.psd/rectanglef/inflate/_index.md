## Inflate(RectangleF, float, float)

Crea y devuelve una copia inflada del especificado[`RectangleF`](../)estructura. La copia se infla en la cantidad especificada. El rectángulo original permanece sin modificar.

```csharp
public static RectangleF Inflate(RectangleF rect, float x, float y)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| rect | RectangleF | El[`RectangleF`](../) para ser copiado. Este rectángulo no se modifica. |
| x | Single | La cantidad para inflar la copia del rectángulo horizontalmente. |
| y | Single | La cantidad para inflar la copia del rectángulo verticalmente. |

### Valor_devuelto

el inflado[`RectangleF`](../).
