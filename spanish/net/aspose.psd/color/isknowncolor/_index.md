## Color.IsKnownColor property

Obtiene un valor que indica si este[`Color`](../) la estructura es un color predefinido. Los colores predefinidos están representados por los elementos del[`KnownColor`](../../knowncolor/) enumeración.

```csharp
public bool IsKnownColor { get; }
```

### El valor de la propiedad

cierto si esto[`Color`](../) se creó a partir de un color predefinido usando el[`FromName`](../fromname/) método o el[`FromKnownColor`](../fromknowncolor/) método; en caso contrario, falso.
