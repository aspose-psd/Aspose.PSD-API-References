## PixelDataFormat.RgbIndexed1Bpp property

Obtiene el[`PixelDataFormat`](../) definido para 1 bit indexado por color. El almacenamiento de datos de píxeles indexados está diseñado para permitir el almacenamiento y la recuperación de datos en cualquier lugar donde se use la paleta de colores. Úselo con precaución, ya que puede requerir la conversión de una paleta a otra o de RGBA a un modelo de color indexado .

```csharp
public static PixelDataFormat RgbIndexed1Bpp { get; }
```

### El valor de la propiedad

El[`PixelDataFormat`](../) definido para indexado 1 bit por color.
