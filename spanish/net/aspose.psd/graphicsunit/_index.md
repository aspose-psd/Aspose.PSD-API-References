## GraphicsUnit enumeration

Especifica la unidad de medida para los datos dados.

```csharp
public enum GraphicsUnit
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| World | `0` | Especifica la unidad del sistema de coordenadas universales como unidad de medida. |
| Display | `1` | Especifica la unidad de medida del dispositivo de visualización. Por lo general, píxeles para pantallas de video y 1/100 de pulgada para impresoras. |
| Pixel | `2` | Especifica un píxel de dispositivo como unidad de medida. |
| Point | `3` | Especifica el punto de una impresora (1/72 pulgadas) como unidad de medida. |
| Inch | `4` | Especifica la pulgada como unidad de medida. |
| Document | `5` | Especifica la unidad del documento (1/300 pulgadas) como unidad de medida. |
| Millimeter | `6` | Especifica el milímetro como unidad de medida. |
