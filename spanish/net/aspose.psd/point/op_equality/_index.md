## Point Equality operator

Compara dos[`Point`](../) objetos. El resultado especifica si los valores de la[`X`](../x/) y[`Y`](../y/) propiedades de los dos[`Point`](../) los objetos son iguales.

```csharp
public static bool operator ==(Point point1, Point point2)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| point1 | Point | Un primero[`Point`](../) comparar. |
| point2 | Point | Un segundo[`Point`](../) comparar. |

### Valor_devuelto

cierto si el[`X`](../x/) y[`Y`](../y/) valores de*point1* y*point2* son iguales; en caso contrario, falso.
