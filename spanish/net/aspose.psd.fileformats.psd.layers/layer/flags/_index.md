## Layer.Flags property

Obtiene o establece los indicadores de capa. bit 0 = transparencia protegida; bit 1 = visible; bit 2 = obsoleto; bit 3 = 1 para Photoshop 5.0 y posterior, indica si el bit 4 tiene información útil; bit 4 = datos de píxeles irrelevantes para la apariencia del documento.

```csharp
public LayerFlags Flags { get; set; }
```

### El valor de la propiedad

Las banderas de la capa.
