## PngFilterType enumeration

El tipo de filtro png.

```csharp
public enum PngFilterType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | `0` | El filtro nulo significa que no se filtran las filas de datos de imagen. |
| Sub | `1` | El subfiltro significa que se aplicará un filtrado sustractivo a los datos de la imagen. |
| Up | `2` | El filtro ascendente significa que se aplicará el filtro de sustracción fila por fila. |
| Avg | `3` | El filtro promedio, significa que ese filtro promedio se aplicará a los datos de la imagen. |
| Paeth | `4` | El filtro predictor de trayectoria. |
| Adaptive | `5` | Filtrado adaptativo, significa que el proceso de guardado elegirá el filtro más adecuado para cada fila de datos. Mejor compresión, tiempo de ejecución más lento. |
