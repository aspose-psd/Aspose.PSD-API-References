El espacio de nombres contiene clases auxiliares y métodos relacionados con XMP.

## Clases

| Clase | Descripción |
| --- | --- |
| [LangAlt](./langalt/) | Representa la alternativa de lenguaje XMP. |
| [Namespaces](./namespaces/) | Contiene espacios de nombres utilizados en el documento RDF. |
| [XmpArray](./xmparray/) | Representa matriz Xmp en[`XmpPackage`](../aspose.psd.xmp/xmppackage/) . TODO: La matriz puede contener datos complejos. |
| [XmpArrayHelper](./xmparrayhelper/) | La clase auxiliar para procesar lógica RDF |
| [XmpElementBase](./xmpelementbase/) | Representa el elemento base xmp que contiene atributos. |
| [XmpHeaderPi](./xmpheaderpi/) | Representa la instrucción de procesamiento del encabezado XMP. |
| [XmpMeta](./xmpmeta/) | Representa xmpmeta. Opcional. El propósito de este elemento es identificar los metadatos XMP dentro del texto XML general que podría contener otros usos no XMP de RDF. |
| [XmpPackage](./xmppackage/) | Define la clase XmpPackage que representa la abstracción base para el paquete XMP. |
| [XmpPackageBaseCollection](./xmppackagebasecollection/) | Representa la colección de[`XmpPackage`](../aspose.psd.xmp/xmppackage/) . |
| [XmpPacketWrapper](./xmppacketwrapper/) | Contiene un paquete xmp serializado que incluye encabezado y tráiler. |
| [XmpRdfRoot](./xmprdfroot/) | Representa el elemento rdf:RDF. Un único paquete XMP se serializará utilizando un único elemento XML rdf:RDF. El contenido del elemento rdf:RDF consistirá únicamente en cero o más elementos rdf:Description. |
| [XmpTrailerPi](./xmptrailerpi/) | Representa la instrucción de procesamiento del tráiler XMP. |
