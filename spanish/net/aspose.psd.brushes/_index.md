El espacio de nombres proporciona clases auxiliares y métodos para trabajar con diferentes tipos de pinceles.

## Clases

| Clase | Descripción |
| --- | --- |
| [HatchBrush](./hatchbrush/) | Define un pincel rectangular con un estilo de sombreado, un color de primer plano y un color de fondo. Esta clase no se puede heredar. |
| [LinearGradientBrush](./lineargradientbrush/) | Encapsula un[`Brush`](../aspose.psd/brush/) con un gradiente lineal. Esta clase no se puede heredar. |
| [LinearGradientBrushBase](./lineargradientbrushbase/) | Representa un[`Brush`](../aspose.psd/brush/) con capacidades de degradado y propiedades apropiadas. |
| [LinearMulticolorGradientBrush](./linearmulticolorgradientbrush/) | Representa un[`Brush`](../aspose.psd/brush/) con degradado lineal definido por múltiples colores y posiciones apropiadas. Esta clase no se puede heredar. |
| [PathGradientBrush](./pathgradientbrush/) | Encapsula un[`Brush`](../aspose.psd/brush/) objeto con un degradado. Esta clase no se puede heredar. |
| [PathGradientBrushBase](./pathgradientbrushbase/) | Representa un[`Brush`](../aspose.psd/brush/) con funcionalidad de gradiente de ruta base. |
| [PathMulticolorGradientBrush](./pathmulticolorgradientbrush/) | Encapsula un[`Brush`](../aspose.psd/brush/) objeto con un degradado. Esta clase no se puede heredar. |
| [SolidBrush](./solidbrush/) | El pincel sólido está diseñado para dibujar continuamente con un color específico. Esta clase no se puede heredar. |
| [TextureBrush](./texturebrush/) | Cada propiedad del[`TextureBrush`](../aspose.psd.brushes/texturebrush/) la clase es un[`Brush`](../aspose.psd/brush/) objeto que usa una imagen para llenar el interior de una forma. Esta clase no se puede heredar. |
| [TransformBrush](./transformbrush/) | A[`Brush`](../aspose.psd/brush/) con capacidades de transformación. |
