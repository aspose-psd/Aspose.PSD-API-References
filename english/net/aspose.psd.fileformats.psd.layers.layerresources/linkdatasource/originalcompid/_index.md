## LinkDataSource.OriginalCompId property

Gets the original ID of the currently selected Comp for the child document, which will be -1 if none are selected. This property gets the original layer Comp selection identifier for Smart Objects. [Layer comps in Smart Objects](https://helpx.adobe.com/photoshop/using/layer-comps.html)

```csharp
public int OriginalCompId { get; }
```

### Property Value

The original ID of the currently selected comp for the child document, which will be -1 if none are selected.
