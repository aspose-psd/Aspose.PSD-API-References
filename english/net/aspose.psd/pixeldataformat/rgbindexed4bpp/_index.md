## PixelDataFormat.RgbIndexed4Bpp property

Gets the [`PixelDataFormat`](../../pixeldataformat) defined for indexed 4 bit per color. The indexed pixel data storage is intended to allow data storage and retrieval everywhere the color palette is used. Use with caution, because may require conversion from one palette to another or from RGBA to indexed color model.
