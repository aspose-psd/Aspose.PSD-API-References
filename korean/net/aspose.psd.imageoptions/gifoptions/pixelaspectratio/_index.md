## GifOptions.PixelAspectRatio property

GIF 픽셀 종횡비를 가져오거나 설정합니다.

```csharp
public byte PixelAspectRatio { get; set; }
```

### 자산 가치

GIF 픽셀 종횡비입니다.

### 비고

픽셀 종횡비 - 원본 이미지에 있는 픽셀 종횡비의 근사치 를 계산하는 데 사용되는 요소입니다. 필드의 값이 0이 아닌 경우 가로 세로 비율 의 근사값은 다음 공식에 따라 계산됩니다. 가로 세로 비율 = (픽셀 가로 세로 비율 + 15) / 64 픽셀 가로 세로 비율은 픽셀' s 높이보다 너비. 이 필드의 값 범위는 4:1의 가장 넓은 픽셀에서 1:4의 가장 높은 픽셀을 1/64 단위로 지정할 수 있습니다. 값 : 0 - 화면 비율 정보가 제공되지 않습니다. 1..255 - 계산에 사용된 값입니다.
