## GifOptions.MaxDiff property

허용되는 최대 픽셀 차이를 가져오거나 설정합니다. 0보다 크면 손실 압축이 사용됩니다. 최적의 손실 압축에 대한 권장 값은 80입니다. 30은 매우 가벼운 압축이고 200은 무겁습니다. 압축 알고리즘의 제한으로 인해 손실이 거의 없을 때 가장 잘 작동합니다. 매우 높은 손실 수준은 많은 이득을 제공하지 않습니다. 허용되는 값의 범위는 [0, 1000]입니다.

```csharp
public int MaxDiff { get; set; }
```
