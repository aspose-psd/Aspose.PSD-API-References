## GifOptions.ColorResolution property

GIF 색상 해상도를 가져오거나 설정합니다.

```csharp
public byte ColorResolution { get; set; }
```

### 자산 가치

색상 해상도입니다.

### 비고

색상 해상도 - 원본 이미지에 사용할 수 있는 기본 색상당 비트 수 에서 1을 뺀 값입니다. 이 값은 그래픽에서 실제로 사용된 색상 수가 아니라 그래픽의 색상이 선택된 전체 팔레트 의 크기를 나타냅니다. 예를 들어, 이 필드의 값이 3이면 원본 이미지의 팔레트에는 이미지를 만드는 데 사용할 수 있는 기본 색상당 4비트가 있습니다. 이 값은 전체 팔레트의 모든 색상을 원본 시스템에서 사용할 수 없는 경우에도 원본 팔레트 의 풍부함을 나타내도록 설정해야 합니다.
