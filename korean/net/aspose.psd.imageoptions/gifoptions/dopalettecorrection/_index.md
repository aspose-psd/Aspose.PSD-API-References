## GifOptions.DoPaletteCorrection property

팔레트 보정 적용 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool DoPaletteCorrection { get; set; }
```

### 자산 가치

`진실` 팔레트 보정이 적용된 경우; 그렇지 않으면,`거짓` .

### 비고

팔레트 보정은 이미지를 GIF로 내보낼 때마다 소스 이미지 색상이 가장 잘 일치하는 팔레트를 만들기 위해 분석 됨을 의미합니다(이미지 팔레트가 존재하지 않거나 옵션에 지정되지 않은 경우). 분석 프로세스는 다소 시간이 걸리지만 출력 이미지는 가장 일치하는 색상 팔레트를 가지며 결과는 시각적으로 더 좋습니다.
