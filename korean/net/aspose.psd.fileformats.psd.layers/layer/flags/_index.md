## Layer.Flags property

레이어 플래그를 가져오거나 설정합니다. 비트 0 = 투명도 보호됨; 비트 1 = 표시됨; 비트 2 = 사용되지 않음; 비트 3 = Photoshop 5.0 이상에서 1, 비트 4에 유용한 정보가 있는지 알려줌; 비트 4 = 문서의 모양과 무관한 픽셀 데이터.

public LayerFlags Flags { get; set; }
