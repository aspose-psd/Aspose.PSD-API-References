## LayerLockType enumeration


### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| None | `0` | 레이어 잠금 없음 |
| LockTransparentPixels | `1` | 부분적으로 레이어 잠그기 - 편집을 레이어의 불투명한 부분으로 제한합니다. 이 옵션은 Photoshop 이전 버전의 투명도 유지 옵션과 동일합니다. |
| LockImagePixels | `2` | 부분적으로 레이어 잠그기 - 페인팅 도구를 사용하여 레이어의 픽셀을 수정할 수 없습니다. |
| LockPosition | `4` | 부분적으로 레이어 잠금 - 레이어의 픽셀이 이동되지 않도록 합니다. |
| LockAll | `7` | layer 의 모든 속성 잠금 |
