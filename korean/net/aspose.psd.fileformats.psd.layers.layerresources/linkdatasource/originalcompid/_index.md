## LinkDataSource.OriginalCompId property

하위 문서에 대해 현재 선택한 Comp의 원래 ID를 가져옵니다. 아무것도 선택하지 않으면 -1이 됩니다. 이 속성은 스마트 개체에 대한 원래 레이어 Comp 선택 식별자를 가져옵니다. [스마트 오브젝트의 레이어 구성 요소](https://helpx.adobe.com/photoshop/using/layer-comps.html)

```csharp
public int OriginalCompId { get; }
```

### 자산 가치

하위 문서에 대해 현재 선택된 구성 요소의 원래 ID이며, 선택된 구성 요소가 없으면 -1이 됩니다.
