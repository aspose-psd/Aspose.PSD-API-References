## 클래스

| 수업 | 설명 |
| --- | --- |
| [LangAlt](./langalt/) | XMP 언어 대안을 나타냅니다. |
| [Namespaces](./namespaces/) | RDF 문서에서 사용되는 네임스페이스를 포함합니다. |
| [XmpArray](./xmparray/) | 는 Xmp 배열을 나타냅니다.[`XmpPackage`](../aspose.psd.xmp/xmppackage/) . TODO: 배열에 복잡한 데이터가 포함될 수 있습니다. |
| [XmpArrayHelper](./xmparrayhelper/) | RDF logic 처리를 위한 헬퍼 클래스 |
| [XmpElementBase](./xmpelementbase/) | 속성을 포함하는 기본 xmp 요소를 나타냅니다. |
| [XmpHeaderPi](./xmpheaderpi/) | XMP 헤더 처리 명령을 나타냅니다. |
| [XmpMeta](./xmpmeta/) | xmpmeta를 나타냅니다. 선택적. 이 요소의 목적은 RDF의 다른 비XMP 사용을 포함할 수 있는 일반 XML 텍스트 내에서 XMP 메타데이터를 식별하는 것입니다. |
| [XmpPackage](./xmppackage/) | XMP 패키지의 기본 추상화를 나타내는 XmpPackage 클래스를 정의합니다. |

| [XmpPacketWrapper](./xmppacketwrapper/) | 헤더와 트레일러를 포함하는 직렬화된 xmp 패키지를 포함합니다. |
| [XmpRdfRoot](./xmprdfroot/) | rdf:RDF 요소를 나타냅니다. 단일 XMP 패킷은 단일 rdf:RDF XML 요소를 사용하여 직렬화됩니다. rdf:RDF 요소 내용은 0개 이상의 rdf:Description 요소로만 구성됩니다. |
| [XmpTrailerPi](./xmptrailerpi/) | XMP 트레일러 처리 명령을 나타냅니다. |
