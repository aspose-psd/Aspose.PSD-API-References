## TransformBrush.IsTransformChanged property

어떤 식으로든 변환이 변경되었는지 여부를 나타내는 값을 가져옵니다. 예를 들어 변환 행렬을 설정하거나 변환 행렬을 변경하는 메서드를 호출합니다. 이 속성은 GDI+. 와의 하위 호환성을 위해 도입되었습니다.

```csharp
public bool IsTransformChanged { get; }
```

### 자산 가치

`진실` 변환이 변경된 경우; 그렇지 않으면,`거짓` .
