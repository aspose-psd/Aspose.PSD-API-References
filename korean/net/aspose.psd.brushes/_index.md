네임스페이스는 다양한 브러시 유형으로 작업할 수 있는 도우미 클래스와 메서드를 제공합니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [HatchBrush](./hatchbrush/) | 해치 스타일, 전경색 및 배경색으로 사각형 브러시를 정의합니다. 이 클래스는 상속될 수 없습니다. |
| [LinearGradientBrush](./lineargradientbrush/) | 캡슐화[`Brush`](../aspose.psd/brush/) 선형 그래디언트로. 이 클래스는 상속될 수 없습니다. |
| [LinearGradientBrushBase](./lineargradientbrushbase/) | 는[`Brush`](../aspose.psd/brush/) 그래디언트 기능과 적절한 속성이 있습니다. |
| [LinearMulticolorGradientBrush](./linearmulticolorgradientbrush/) | 는[`Brush`](../aspose.psd/brush/) 여러 색상과 적절한 위치로 정의된 선형 그래디언트가 있습니다. 이 클래스는 상속될 수 없습니다. |
| [PathGradientBrush](./pathgradientbrush/) | 캡슐화[`Brush`](../aspose.psd/brush/) 그라디언트가 있는 개체입니다. 이 클래스는 상속될 수 없습니다. |
| [PathGradientBrushBase](./pathgradientbrushbase/) | 는[`Brush`](../aspose.psd/brush/) 기본 경로 그래디언트 기능 포함. |
| [PathMulticolorGradientBrush](./pathmulticolorgradientbrush/) | 캡슐화[`Brush`](../aspose.psd/brush/) 그라디언트가 있는 개체입니다. 이 클래스는 상속될 수 없습니다. |
| [SolidBrush](./solidbrush/) | 단색 브러시는 특정 색상으로 연속적으로 그리기 위한 것입니다. 이 클래스는 상속될 수 없습니다. |
| [TextureBrush](./texturebrush/) | 각 속성은[`TextureBrush`](../aspose.psd.brushes/texturebrush/) 수업은[`Brush`](../aspose.psd/brush/) 도형의 내부를 채우기 위해 이미지를 사용하는 객체. 이 클래스는 상속될 수 없습니다. |
| [TransformBrush](./transformbrush/) | A[`Brush`](../aspose.psd/brush/) 변환 기능이 있습니다. |
