## ExifData.ISOSpeedLatitudeYYY property

ISO 12232에 정의된 카메라 또는 입력 장치의 ISO 속도 위도 yyy 값을 가져오거나 설정합니다.

```csharp
public uint ISOSpeedLatitudeYYY { get; set; }
```

### 자산 가치

ISO 12232에 정의된 카메라 또는 입력 장치의 ISO 감도 위도 yyy 값입니다.

### 비고

이 태그는 ISOSpeed 및 ISOSpeedLatitudezzz 없이 기록되지 않습니다.
