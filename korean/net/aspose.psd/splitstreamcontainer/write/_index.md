## Write(byte[], int, int) {#write_1}

바이트 시퀀스를 현재 스트림에 쓰고 이 스트림 내에서 현재 위치를 쓴 바이트 수만큼 앞으로 이동합니다.

```csharp
public override void Write(byte[] buffer, int offset, int count)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| buffer | Byte[] | 바이트 배열입니다. 이 방법은 복사*count* 바이트*buffer* 현재 스트림으로. |
| offset | Int32 | 0부터 시작하는 바이트 오프셋*buffer*현재 스트림에 바이트를 복사하기 시작하는 시점입니다. |
| count | Int32 | 현재 스트림에 쓸 바이트 수입니다. |
