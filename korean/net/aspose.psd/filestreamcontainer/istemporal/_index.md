## FileStreamContainer.IsTemporal property

스트림이 일시적인지 여부를 나타내는 값을 가져오거나 설정합니다.

```csharp
public bool IsTemporal { get; set; }
```

### 자산 가치

`진실` 스트림이 일시적인 경우; 그렇지 않으면,`거짓` .

### 비고

일시적인 흐름은 처분되면 스스로 제거됩니다. 스트림이 메모리 기반인 경우 이 속성은 영향을 미치지 않습니다. 스트림이 명시적으로 생성된 경우 스트림을 임시 또는 지속으로 표시할 수 있습니다. 그렇지 않으면 적절한 예외가 발생합니다.
