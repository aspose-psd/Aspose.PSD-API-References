## ToCmykIcc(Color[], Stream, Stream) {#tocmykicc_3}

사용자 정의 프로파일과 함께 Icc 변환을 사용하여 ARGB 색상에서 CMYK 색상으로 변환.

```csharp
public static int[] ToCmykIcc(Color[] pixels, Stream rgbIccStream, Stream cmykIccStream)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| pixels | Color[] | ARGB 색상. |
| rgbIccStream | Stream | RGB Icc 프로파일을 포함하는 스트림. |
| cmykIccStream | Stream | CMYK Icc 프로파일을 포함하는 스트림. |

### 반환 값

32비트 정수 값으로 표시되는 CMYK 색상입니다.
