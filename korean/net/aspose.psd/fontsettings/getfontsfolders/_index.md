## FontSettings.GetFontsFolders method

Aspose.Words가 트루타입 글꼴을 찾는 폴더 목록이 포함된 배열의 복사본을 가져옵니다.

```csharp
public static string[] GetFontsFolders()
```

### 반환 값

현재 글꼴 위치의 사본.

### 비고

반환된 값은 Aspose.Words가 사용하는 데이터의 복사본입니다. 반환된 배열의 항목을 변경하면 문서 렌더링에 영향을 주지 않습니다. 새 글꼴 위치를 지정하려면[`SetFontsFolders`](../setfontsfolders/) 방법.
