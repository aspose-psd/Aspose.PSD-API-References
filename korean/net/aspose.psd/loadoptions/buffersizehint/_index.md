## LoadOptions.BufferSizeHint property

모든 내부 버퍼에 대해 정의된 최대 허용 크기인 버퍼 크기 힌트를 가져오거나 설정합니다.

```csharp
public int BufferSizeHint { get; set; }
```

### 자산 가치

버퍼 크기 힌트(MB). 비양수 값은 내부 버퍼에 대한 메모리 제한이 없음을 의미합니다.
