## Brush.Opacity property

브러시 불투명도를 가져오거나 설정합니다. 값은 0과 1 사이여야 합니다. 값이 0이면 브러시가 완전히 표시되고 값이 1이면 브러시가 완전히 불투명합니다.

```csharp
public float Opacity { get; set; }
```

### 자산 가치
