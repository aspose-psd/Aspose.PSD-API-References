## Pen.SetLineCap method

```csharp
public void SetLineCap(LineCap startCap, LineCap endCap, DashCap dashCap)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| startCap | LineCap | ㅏ[`LineCap`](../../linecap/) 이것으로 그린 선의 시작 부분에 사용할 캡 스타일을 나타냅니다.[`Pen`](../). |

| dashCap | DashCap | ㅏ[`LineCap`](../../linecap/) 이것은 이것으로 그린 점선의 시작 또는 끝에서 사용할 캡 스타일을 나타냅니다.[`Pen`](../). |
