## StringFormat.DigitSubstitutionLanguage property

지역 숫자가 서부 숫자로 대체될 때 사용되는 언어를 가져오거나 설정합니다.

```csharp
public int DigitSubstitutionLanguage { get; set; }
```

### 자산 가치

현지 숫자가 서부 숫자로 대체될 때 사용될 언어를 식별하는 NLS(National Language Support) 언어 식별자입니다. 다음을 전달할 수 있습니다.LCID 재산CultureInfo 객체를 NLS 언어 식별자로 사용합니다. 예를 들어,CultureInfo 문자열 "ar-EG"를CultureInfo constructor. 를 전달하면LCID 그 재산CultureInfo . 와 함께 객체Traditional ~로StringDigitSubstitute) method, 표시 시간에 아라비아-인도 숫자가 서부 숫자로 대체됩니다.

### 비고

더 이상 사용되지 않는 SetDigitSubstitution 메서드에 대해 setter가 도입되었습니다.
