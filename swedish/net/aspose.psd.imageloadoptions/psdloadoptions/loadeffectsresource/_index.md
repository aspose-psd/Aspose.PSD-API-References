## PsdLoadOptions.LoadEffectsResource property

Hämtar eller ställer in ett värde som indikerar om [lasteffektresurs] (som standard resurs inte laddas). När det här alternativet är inställt kommer endast effekter som stöds att renderas till den slutliga sammanslagna bilden.

```csharp
public bool LoadEffectsResource { get; set; }
```

### Fastighetsvärde

`Sann` if [lasteffektresurs]; annat,`falsk` .
