## PsdLoadOptions.ReadOnlyMode property

Hämtar eller ställer in ett värde som anger om [använd skrivskyddat läge]. Detta är skrivskyddat läge som stöds för identisk kompatibilitet med Adobe Photoshop. När det här alternativet är inställt sparas inte alla ändringar som tillämpas för lager i den slutliga bilden. All data används från ImageData-sektionen, så den är identisk med Photoshop. Som standard är alla inlästa bilder inte identiska med Adobe Photoshop-kompatibla.

```csharp
public bool ReadOnlyMode { get; set; }
```

### Fastighetsvärde

`Sann` om [använd Photoshop-kompatibilitetsläge]; annat,`falsk` .
