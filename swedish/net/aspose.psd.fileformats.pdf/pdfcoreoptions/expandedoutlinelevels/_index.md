## PdfCoreOptions.ExpandedOutlineLevels property

Anger hur många nivåer i dokumentkonturen som ska visas expanderade när PDF-filen visas. 0 - dokumentkonturen expanderas inte. 1 - objekt på första nivån i dokumentet expanderas och så vidare. Standard är 0.

```csharp
public int ExpandedOutlineLevels { get; set; }
```
