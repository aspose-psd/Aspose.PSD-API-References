## PsdOptions.RemoveGlobalTextEngineResource property

Hämtar eller ställer in ett värde som anger om - Ta bort den globala textmotorresursen - Används för vissa psd-filer med textskikt, endast i fallet när de inte kan öppnas i Adobe Photoshop efter bearbetning (mest för textlager som saknas teckensnitt). Efter att ha använt det här alternativet måste användaren göra nästa i öppnad i Photoshop-fil: Meny "Text" -&gt; "Bearbeta frånvarande teckensnitt". Efter den operationen kommer all text att visas igen. Observera att denna operation kan orsaka några slutliga layoutändringar.

```csharp
public bool RemoveGlobalTextEngineResource { get; set; }
```

### Fastighetsvärde

`Sann` if [ta bort global textmotorresurs]; annat,`falsk` .
