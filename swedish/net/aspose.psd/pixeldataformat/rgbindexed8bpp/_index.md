## PixelDataFormat.RgbIndexed8Bpp property

Får[`PixelDataFormat`](../) definierad för indexerad 8 bitar per färg. Den indexerade pixeldatalagringen är avsedd att möjliggöra datalagring och hämtning överallt där färgpaletten används. Använd med försiktighet, eftersom det kan kräva konvertering från en palett till en annan eller från RGBA till indexerad färgmodell .

public static PixelDataFormat RgbIndexed8Bpp { get; }
