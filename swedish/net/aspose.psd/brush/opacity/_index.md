## Brush.Opacity property

Hämtar eller ställer in borstens opacitet. Värdet ska vara mellan 0 och 1. Värdet 0 betyder att borsten är helt synlig, värdet 1 betyder att borsten är helt ogenomskinlig.

```csharp
public float Opacity { get; set; }
```

### Fastighetsvärde

Borstens opacitetsvärde.
