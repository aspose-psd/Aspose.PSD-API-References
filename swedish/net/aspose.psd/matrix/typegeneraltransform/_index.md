## Matrix.TypeGeneralTransform field

Denna konstant indikerar att transformationen som definieras av detta objekt utför en godtycklig omvandling av de ingående koordinaterna. Om denna transformation kan klassificeras av någon av ovanstående konstanter, kommer typen antingen att vara konstant TypeIdentity eller a kombination av lämplig flagga bitar för de olika koordinat -omvandlingarna som denna transformation utför.

```csharp
public const int TypeGeneralTransform;
```
