## Point Inequality operator

Jämför två[`Point`](../) föremål. Resultatet anger om värdena för[`X`](../x/) eller[`Y`](../y/) egenskaper hos de två[`Point`](../) objekt är ojämlika.

```csharp
public static bool operator !=(Point point1, Point point2)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| point1 | Point | En första[`Point`](../) att jämföra. |
| point2 | Point | En sekund[`Point`](../) att jämföra. |

### Returvärde

Sant om värdena för antingen[`X`](../x/) fastigheter eller[`Y`](../y/) egenskaper hos*point1* och*point2* skilja sig; annars falskt.
