## Point Equality operator

Jämför två[`Point`](../) föremål. Resultatet anger om värdena för[`X`](../x/) och[`Y`](../y/) egenskaper hos de två[`Point`](../) objekt är lika.

```csharp
public static bool operator ==(Point point1, Point point2)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| point1 | Point | En första[`Point`](../) att jämföra. |
| point2 | Point | En sekund[`Point`](../) att jämföra. |

### Returvärde

Sant om[`X`](../x/) och[`Y`](../y/) värden av*point1* och*point2* är jämlika; annars falskt.
