## LinkDataSource.AssetLockedState property

Hämtar eller ställer in ett värde som indikerar om PSD-tillgången är låst. Tillståndet för tillgången är låst, för Adobe® Photoshop® СС Libraries-tillgångar.

```csharp
public bool AssetLockedState { get; set; }
```

### Fastighetsvärde

`Sann` om Adobe® Photoshop®-tillgången är låst; annat,`falsk` .
