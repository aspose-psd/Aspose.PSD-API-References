## LinkDataSource.OriginalCompId property

Hämtar det ursprungliga ID:t för den för närvarande valda Comp för det underordnade dokumentet, vilket kommer att vara -1 om ingen har valts. Den här egenskapen får den ursprungliga lager Comp-urvalsidentifieraren för Smart Objects. [Layer comps i Smart Objects](https://helpx.adobe.com/photoshop/using/layer-comps.html)

```csharp
public int OriginalCompId { get; }
```

### Fastighetsvärde

Det ursprungliga ID:t för den för närvarande valda kompositionen för det underordnade dokumentet, vilket kommer att vara -1 om ingen har valts.
