## ImageOptionsBase.BufferSizeHint property

Λαμβάνει ή ορίζει την υπόδειξη μεγέθους buffer που ορίζεται ως μέγιστο επιτρεπόμενο μέγεθος για όλα τα εσωτερικά buffer.

```csharp
public int BufferSizeHint { get; set; }
```

### Αξία περιουσίας

Η υπόδειξη μεγέθους buffer, σε megabyte. Η μη θετική τιμή σημαίνει ότι δεν υπάρχει περιορισμός μνήμης για εσωτερικά buffers
