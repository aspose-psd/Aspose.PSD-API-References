## DataRecoveryMode enumeration

Η λειτουργία ανάκτησης δεδομένων.

```csharp
public enum DataRecoveryMode
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| None | `0` | Δεν υπονοείται ανάκτηση δεδομένων. Κάθε φορά που η μορφή αρχείου έχει κάποια κατεστραμμένα δεδομένα, δημιουργείται η κατάλληλη εξαίρεση. |
| ConsistentRecover | `1` | Η συνεπής λειτουργία ανάκτησης προσπαθεί να ανακτήσει όλα τα δεδομένα, εφόσον η καταστροφή δεν παραβιάζει τη μορφή αρχείου και επιτρέπει τη σωστή περαιτέρω επεξεργασία. |
| MaximalRecover | `2` | Η λειτουργία μέγιστης ανάκτησης ανακτά όλα τα δεδομένα ακόμα κι αν η μορφή του αρχείου έχει καταστραφεί και η περαιτέρω επεξεργασία μπορεί να αποφέρει αποτελέσματα χωρίς παρακολούθηση. |
