## InterpolationMode enumeration

Το`InterpolationMode` Η απαρίθμηση καθορίζει τον αλγόριθμο που χρησιμοποιείται όταν οι εικόνες κλιμακώνονται ή περιστρέφονται.

```csharp
public enum InterpolationMode
```

### Αξίες

| Ονομα | αξία | Περιγραφή |
| --- | --- | --- |
| Invalid | `-1` | Μη έγκυρη λειτουργία παρεμβολής. |
| Default | `0` | Καθορίζει την προεπιλεγμένη λειτουργία. |
| Low | `1` | Καθορίζει παρεμβολή χαμηλής ποιότητας. |
| High | `2` | Καθορίζει παρεμβολή υψηλής ποιότητας. |
| Bilinear | `3` | Καθορίζει τη διγραμμική παρεμβολή. Δεν γίνεται προφιλτράρισμα. Αυτή η λειτουργία δεν είναι κατάλληλη για τη συρρίκνωση μιας εικόνας κάτω από το 50 τοις εκατό του αρχικού της μεγέθους. |
| Bicubic | `4` | Καθορίζει τη δικυβική παρεμβολή. Δεν γίνεται προφιλτράρισμα. Αυτή η λειτουργία δεν είναι κατάλληλη για τη συρρίκνωση μιας εικόνας κάτω από το 25 τοις εκατό του αρχικού της μεγέθους. |
| NearestNeighbor | `5` | Καθορίζει την παρεμβολή πλησιέστερου γείτονα. |
| HighQualityBilinear | `6` | Καθορίζει υψηλής ποιότητας, διγραμμική παρεμβολή. Εκτελείται προφιλτράρισμα για να διασφαλιστεί συρρίκνωση υψηλής ποιότητας. |
| HighQualityBicubic | `7` | Καθορίζει δικυβική παρεμβολή υψηλής ποιότητας. Το προφιλτράρισμα πραγματοποιείται για να εξασφαλιστεί συρρίκνωση υψηλής ποιότητας. Αυτή η λειτουργία παράγει μεταμορφωμένες εικόνες υψηλότερης ποιότητας. |
