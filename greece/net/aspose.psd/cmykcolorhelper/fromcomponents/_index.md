## CmykColorHelper.FromComponents method

Δημιουργεί CMYK από τιμές κυανό, ματζέντα, κίτρινο και μαύρο 32 bit.

```csharp
public static int FromComponents(int cyan, int magenta, int yellow, int black)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| cyan | Int32 | Το κυανό συστατικό. Οι έγκυρες τιμές είναι από 0 έως 255. |
| magenta | Int32 | Το συστατικό ματζέντα. Οι έγκυρες τιμές είναι από 0 έως 255. |
| yellow | Int32 | Το κίτρινο συστατικό. Οι έγκυρες τιμές είναι από 0 έως 255. |
| black | Int32 | Το μαύρο συστατικό. Οι έγκυρες τιμές είναι από 0 έως 255. |

### Επιστρεφόμενη Αξία

Το χρώμα CMYK παρουσιάζεται ως ακέραιος αριθμός 32 bit.
