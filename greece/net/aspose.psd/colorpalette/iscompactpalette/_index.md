## ColorPalette.IsCompactPalette property

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν χρησιμοποιείται συμπαγής παλέτα.

```csharp
public bool IsCompactPalette { get; }
```

### Αξία περιουσίας

`αληθής` εάν χρησιμοποιείται συμπαγής παλέτα. σε διαφορετική περίπτωση,`ψευδής`.

### Παρατηρήσεις

Η συμπαγής παλέτα σημαίνει ότι η εικόνα θα περιέχει μόνο τις καθορισμένες εγγραφές παλέτας εάν είναι δυνατόν ή με άλλα λόγια η εικόνα θα είναι πιο συμπαγής και θα καταλαμβάνει λιγότερο χώρο· διαφορετικά θα υπάρχουν καταχωρήσεις 2^BitsPerPixel και η εικόνα θα διατηρεί περισσότερο χώρο για όλες τις πιθανές καταχωρήσεις παλέτας. Η ρύθμιση αυτής της τιμής σε true και η αλλαγή των καταχωρήσεων παλέτας μπορεί να προκαλέσει ποινή απόδοσης, καθώς μπορεί να συμβεί μετακίνηση δεδομένων, γι' αυτό χρησιμοποιήστε το προσεκτικά.
