## TransparencySupporter.Opacity property

Λαμβάνει ή ορίζει την αδιαφάνεια του αντικειμένου. Η τιμή πρέπει να είναι μεταξύ 0 και 1. Η τιμή 0 σημαίνει ότι το αντικείμενο είναι πλήρως ορατό, η τιμή 1 σημαίνει ότι το αντικείμενο είναι πλήρως αδιαφανές.

```csharp
public float Opacity { get; set; }
```

Η τιμή της αδιαφάνειας.
