## Brush.Opacity property

Λαμβάνει ή ρυθμίζει την αδιαφάνεια του πινέλου. Η τιμή πρέπει να είναι μεταξύ 0 και 1. Η τιμή 0 σημαίνει ότι η βούρτσα είναι πλήρως ορατή, η τιμή 1 σημαίνει ότι η βούρτσα είναι πλήρως αδιαφανής.

```csharp
public float Opacity { get; set; }
```

### Αξία περιουσίας

Η τιμή αδιαφάνειας του πινέλου.
