## Matrix.TypeGeneralTransform field

Αυτή η σταθερά δείχνει ότι ο μετασχηματισμός που ορίζεται από αυτό το αντικείμενο εκτελεί μια αυθαίρετη μετατροπή των συντεταγμένων εισόδου. Εάν αυτός ο μετασχηματισμός μπορεί να ταξινομηθεί με οποιαδήποτε από τις παραπάνω σταθερές, ο τύπος θα είναι είτε ο σταθερός TypeIdentity είτε ο συνδυασμός a της κατάλληλης σημαίας bit για τις διάφορες συντεταγμένες μετατροπές που εκτελεί αυτός ο μετασχηματισμός.

```csharp
public const int TypeGeneralTransform;
```
