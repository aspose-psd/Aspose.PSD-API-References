## Matrix.TypeIdentity field

Ένας μετασχηματισμός ταυτότητας είναι αυτός στον οποίο οι συντεταγμένες εξόδου είναι πάντα ίδιες με τις συντεταγμένες εισόδου. Εάν αυτός ο μετασχηματισμός είναι κάτι διαφορετικό από τον μετασχηματισμό ταυτότητας, ο τύπος θα είναι είτε η σταθερά GENERAL_TRANSFORM είτε a συνδυασμός των κατάλληλων bits σημαίας για τις διάφορες συντεταγμένες μετατροπές που εκτελεί αυτός ο μετασχηματισμός.

```csharp
public const int TypeIdentity;
```
