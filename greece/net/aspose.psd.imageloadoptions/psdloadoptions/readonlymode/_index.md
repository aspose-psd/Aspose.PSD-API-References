## PsdLoadOptions.ReadOnlyMode property

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν [χρήση λειτουργίας μόνο για ανάγνωση]. Αυτή είναι η λειτουργία μόνο για ανάγνωση, που υποστηρίζεται για πανομοιότυπη συμβατότητα με το Adobe Photoshop. Όταν οριστεί αυτή η επιλογή, όλες οι αλλαγές που εφαρμόζονται για τα επίπεδα δεν θα αποθηκευτούν στην τελική εικόνα. Όλα τα δεδομένα χρησιμοποιούνται από την ενότητα ImageData, επομένως είναι πανομοιότυπα με το Photoshop. Από προεπιλογή, όλες οι εικόνες που έχουν φορτωθεί δεν είναι πανομοιότυπες με τις συμβατές με το Adobe Photoshop.

```csharp
public bool ReadOnlyMode { get; set; }
```

### Αξία περιουσίας

`αληθής` εάν [χρήση λειτουργίας συμβατότητας photoshop]; σε διαφορετική περίπτωση,`ψευδής` .
