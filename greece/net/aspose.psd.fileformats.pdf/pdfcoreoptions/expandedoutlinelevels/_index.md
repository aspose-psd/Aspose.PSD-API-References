## PdfCoreOptions.ExpandedOutlineLevels property

Καθορίζει πόσα επίπεδα στο περίγραμμα του εγγράφου θα εμφανίζονται διευρυμένα όταν προβάλλεται το αρχείο PDF. 0 - το περίγραμμα του εγγράφου δεν επεκτείνεται. 1 - τα στοιχεία πρώτου επιπέδου στο έγγραφο επεκτείνονται και ούτω καθεξής. Η προεπιλογή είναι 0.

```csharp
public int ExpandedOutlineLevels { get; set; }
```
