Ο χώρος ονομάτων περιέχει διαφορετικά σχήματα συνδυασμένα από τμήματα σχήματος.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [ArcShape](./arcshape/) | Αντιπροσωπεύει ένα σχήμα τόξου. |
| [BezierShape](./beziershape/) | Αντιπροσωπεύει έναν spline bezier. |
| [CurveShape](./curveshape/) | Αντιπροσωπεύει ένα σχήμα καμπύλης σφήνας. |
| [EllipseShape](./ellipseshape/) | Αντιπροσωπεύει ένα σχήμα έλλειψης. |
| [PieShape](./pieshape/) | Αντιπροσωπεύει σχήμα πίτας. |
| [PolygonShape](./polygonshape/) | Αντιπροσωπεύει σχήμα πολυγώνου. |
| [RectangleProjectedShape](./rectangleprojectedshape/) | Αντιπροσωπεύει ένα σχήμα που προβάλλεται πάνω από ορθογώνιο γυρισμένο σε συγκεκριμένο προσανατολισμό. Καθορίζεται από τέσσερα σημεία που μπορούν να περιστραφούν στο διάστημα διατηρώντας το ίδιο μήκος άκρων και 90 μοίρες μεταξύ των γειτονικών άκρων. |
| [RectangleShape](./rectangleshape/) | Αντιπροσωπεύει ένα ορθογώνιο σχήμα. |
| [TextShape](./textshape/) | Αντιπροσωπεύει ένα σχήμα κειμένου. |
