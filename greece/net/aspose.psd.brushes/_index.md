Ο χώρος ονομάτων παρέχει βοηθητικές κλάσεις και μεθόδους για εργασία με διαφορετικούς τύπους πινέλων.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [HatchBrush](./hatchbrush/) | Καθορίζει ένα ορθογώνιο πινέλο με στυλ καταπακτής, χρώμα προσκηνίου και χρώμα φόντου. Αυτή η κλάση δεν μπορεί να κληρονομηθεί. |
| [LinearGradientBrush](./lineargradientbrush/) | Ενθυλακώνει α[`Brush`](../aspose.psd/brush/) με γραμμική κλίση. Αυτή η κλάση δεν μπορεί να κληρονομηθεί. |
| [LinearGradientBrushBase](./lineargradientbrushbase/) | Αντιπροσωπεύει α[`Brush`](../aspose.psd/brush/) με δυνατότητες κλίσης και κατάλληλες ιδιότητες. |
| [LinearMulticolorGradientBrush](./linearmulticolorgradientbrush/) | Αντιπροσωπεύει α[`Brush`](../aspose.psd/brush/) με γραμμική κλίση που ορίζεται από πολλαπλά χρώματα και κατάλληλες θέσεις. Αυτή η κλάση δεν μπορεί να κληρονομηθεί. |
| [PathGradientBrush](./pathgradientbrush/) | Ενθυλακώνει α[`Brush`](../aspose.psd/brush/) αντικείμενο με κλίση. Αυτή η κλάση δεν μπορεί να κληρονομηθεί. |
| [PathGradientBrushBase](./pathgradientbrushbase/) | Αντιπροσωπεύει α[`Brush`](../aspose.psd/brush/) με λειτουργικότητα κλίσης διαδρομής βάσης. |
| [PathMulticolorGradientBrush](./pathmulticolorgradientbrush/) | Ενθυλακώνει α[`Brush`](../aspose.psd/brush/) αντικείμενο με κλίση. Αυτή η κλάση δεν μπορεί να κληρονομηθεί. |
| [SolidBrush](./solidbrush/) | Το Το συμπαγές πινέλο προορίζεται για συνεχή σχεδίαση με συγκεκριμένο χρώμα. Αυτή η κλάση δεν μπορεί να κληρονομηθεί. |
| [TextureBrush](./texturebrush/) | Κάθε ιδιότητα του[`TextureBrush`](../aspose.psd.brushes/texturebrush/) η τάξη είναι α[`Brush`](../aspose.psd/brush/) αντικείμενο που χρησιμοποιεί μια εικόνα για να γεμίσει το εσωτερικό ενός σχήματος. Αυτή η κλάση δεν μπορεί να κληρονομηθεί. |
| [TransformBrush](./transformbrush/) | Α[`Brush`](../aspose.psd/brush/) με δυνατότητες μετασχηματισμού. |
