## LinkDataSource.AssetLockedState property

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν το στοιχείο PSD είναι κλειδωμένο. Η κατάσταση κλειδώματος στοιχείου, για τα στοιχεία του Adobe® Photoshop® СС Libraries.

```csharp
public bool AssetLockedState { get; set; }
```

### Αξία περιουσίας

`αληθής` εάν το στοιχείο Adobe® Photoshop® είναι κλειδωμένο. σε διαφορετική περίπτωση,`ψευδής` .
