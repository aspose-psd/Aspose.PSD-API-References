## LinkDataSource.ChildDocId property

Λαμβάνει ή ορίζει το αναγνωριστικό θυγατρικού εγγράφου στην πηγή δεδομένων liFE ή liFD του πόρου Lnk2 / LnkE Adobe® Photoshop®.

```csharp
public string ChildDocId { get; set; }
```

### Αξία περιουσίας

Το αναγνωριστικό θυγατρικού εγγράφου στην πηγή δεδομένων liFE ή liFD του πόρου PSD Lnk2 / LnkE.
