## Layer.LayerLock property

Λαμβάνει ή ορίζει το κλείδωμα επιπέδου. Λάβετε υπόψη ότι εάν έχει οριστεί η σημαία LayerFlags.TransparencyProtected, θα αντικατασταθεί από τη σημαία κλειδώματος επιπέδου. Για να επιστρέψετε τη σημαία LayerFlags.TransparencyProtected πρέπει να κάνετε αίτηση για την επιλογή layer.Flagsparency00.

public LayerLockType LayerLock { get; set; }
