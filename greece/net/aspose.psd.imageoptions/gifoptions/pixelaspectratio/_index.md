## GifOptions.PixelAspectRatio property

Λαμβάνει ή ορίζει την αναλογία διάστασης εικονοστοιχείων GIF.

```csharp
public byte PixelAspectRatio { get; set; }
```

### Αξία περιουσίας

Η αναλογία εικονοστοιχείων GIF.

### Παρατηρήσεις

Λόγος διαστάσεων εικονοστοιχείου - Συντελεστής που χρησιμοποιείται για τον υπολογισμό κατά προσέγγιση του λόγου διαστάσεων του εικονοστοιχείου στην αρχική εικόνα. Εάν η τιμή του πεδίου δεν είναι 0, αυτή η προσέγγιση του λόγου διαστάσεων υπολογίζεται με βάση τον τύπο: Αναλογία διαστάσεων = (Pixel Aspect Ratio + 15) / 64 Η αναλογία διαστάσεων pixel ορίζεται ως το πηλίκο του pixel Aspect Ratio s πλάτος στο ύψος του. Το εύρος τιμών σε αυτό το πεδίο επιτρέπει προδιαγραφή του ευρύτερου εικονοστοιχείου από 4:1 έως το ψηλότερο εικονοστοιχείο από 1:4 σε προσαυξήσεις του 1/64ου. Τιμές : 0 - Δεν δίνονται πληροφορίες αναλογίας διαστάσεων._x0.25_5_. Τιμή που χρησιμοποιείται στον υπολογισμό.
