## GifOptions.DoPaletteCorrection property

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν εφαρμόζεται διόρθωση παλέτας.

```csharp
public bool DoPaletteCorrection { get; set; }
```

### Αξία περιουσίας

`αληθής` εάν εφαρμόζεται διόρθωση παλέτας. σε διαφορετική περίπτωση,`ψευδής` .

### Παρατηρήσεις

Η διόρθωση παλέτας σημαίνει ότι κάθε φορά που η εικόνα εξάγεται σε GIF, τα χρώματα της αρχικής εικόνας θα αναλυθούν προκειμένου να δημιουργηθεί η καλύτερη αντίστοιχη παλέτα (σε περίπτωση που η παλέτα εικόνων δεν υπάρχει ή δεν καθορίζεται στις επιλογές). Η διαδικασία ανάλυσης διαρκεί κάποιο χρόνο, ωστόσο η Η εικόνα εξόδου θα έχει την καλύτερη αντίστοιχη παλέτα χρωμάτων και το αποτέλεσμα είναι οπτικά καλύτερο.
