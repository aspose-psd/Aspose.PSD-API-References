## Aspose.PSD for .NET

{{% alert color="primary" %}} 

![Aspose.PSD for .NET 製品ロゴ](../home_1.png)


Aspose.PSD for .NET は、PSD ファイル フォーマットの広範な操作を可能にする製品です。 この製品は、Adobe Photoshop をインストールする必要はありません。 Aspose.PSD for .NET を使用すると、PSD ファイルの編集、レイヤー プロパティの更新、透かしの追加、グラフィック操作の実行、ファイル フォーマットの別のフォーマットへの変換を行うことができます。 この製品を使用すると、開発者を支援するさまざまな自動化シナリオが可能になります。

現在、Aspose.PSD for .NET は、読み込みと操作のために PSD および PSB ファイル フォーマットをサポートしています。 将来のバージョンでは、より多くの PSD ファイル形式をロードできるようになります。 Aspose.PSD for .NET では、TIFF、JPEG、JPEG2000、PNG、GIF、BMP などのさまざまなラスター ファイル フォーマットへのエクスポートが可能で、エクスポートでサポートされるファイル フォーマットがさらに追加される予定です。 この製品は、効率的な方法で PSD ファイルの編集を実行し、優れたパフォーマンスを維持できるアルゴリズムを使用しています。 この製品は積極的に開発されており、市場の要求をサポートするために多くの便利な機能が追加されます。 たとえば、.NET Core のサポートは積極的に開発されています。

## Aspose.PSD for Java

{{% alert color="primary" %}} 

![Aspose.PSD for Java 製品ロゴ](../home_2.png)


Aspose.PSD for Java では、PSD ファイル フォーマットを幅広く操作できます。 この製品は、Adobe Photoshop をインストールする必要はありません。 Aspose.PSD for Java を使用すると、PSD ファイルの編集、レイヤー プロパティの更新、透かしの追加、グラフィック操作の実行、あるファイル フォーマットを別のファイル フォーマットに変換することができます。 この製品は、開発者を支援するさまざまな自動化シナリオもサポートしています。

現在、Aspose.PSD for Java は、ロードおよび操作用に PSD および PSB ファイル フォーマットをサポートしています。 将来のバージョンでは、より多くの PSD ファイル形式をロードできるようになります。 Aspose.PSD for Java を使用すると、TIFF、JPEG、JPEG2000、PNG、GIF、BMP などのさまざまなラスター ファイル フォーマットにエクスポートできます。 この製品は、効率的な方法で PSD ファイルの編集を実行し、優れたパフォーマンスを維持できるアルゴリズムを使用しています。 この製品は積極的に開発されており、市場の要求を満たすために多くの便利な機能が追加されます。
