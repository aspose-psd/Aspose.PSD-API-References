## GlobalLayerMaskInfo.Kind property

種類を取得または設定します。 0 = 色を選択、つまり反転; 1 = 色を保護; 128 = レイヤーごとに保存された値を使用。この値が推奨されます. 他の値は、ベータ版との後方互換性のためのものです.

```csharp
public byte Kind { get; set; }
```
