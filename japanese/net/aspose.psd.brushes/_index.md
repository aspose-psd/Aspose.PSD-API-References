名前空間は、さまざまな種類のブラシを操作するためのヘルパー クラスとメソッドを提供します。

## クラス

| クラス | 説明 |
| --- | --- |
| [HatchBrush](./hatchbrush/) | ハッチング スタイル、前景色、および背景色を持つ長方形のブラシを定義します。このクラスは継承できません. |
| [LinearGradientBrush](./lineargradientbrush/) | をカプセル化します[`Brush`](../aspose.psd/brush/)線形グラデーションで。このクラスは継承できません. |
| [LinearGradientBrushBase](./lineargradientbrushbase/) | を表します[`Brush`](../aspose.psd/brush/)グラデーション機能と適切なプロパティを使用. |
| [LinearMulticolorGradientBrush](./linearmulticolorgradientbrush/) | を表します[`Brush`](../aspose.psd/brush/)複数の色と適切な位置で定義された線形グラデーション。このクラスは継承できません. |
| [PathGradientBrush](./pathgradientbrush/) | をカプセル化します[`Brush`](../aspose.psd/brush/)グラデーションのあるオブジェクト。このクラスは継承できません. |
| [PathGradientBrushBase](./pathgradientbrushbase/) | を表します[`Brush`](../aspose.psd/brush/)ベースパスグラデーション機能付き. |
| [PathMulticolorGradientBrush](./pathmulticolorgradientbrush/) | をカプセル化します[`Brush`](../aspose.psd/brush/)グラデーションのあるオブジェクト。このクラスは継承できません. |
| [SolidBrush](./solidbrush/) | ソリッド ブラシは、特定の色で連続して描画することを目的としています。このクラスは継承できません. |
| [TextureBrush](./texturebrush/) | の各プロパティ[`TextureBrush`](../aspose.psd.brushes/texturebrush/)クラスは[`Brush`](../aspose.psd/brush/)イメージを使用して形状の内部を塗りつぶすオブジェクト。このクラスは継承できません. |
| [TransformBrush](./transformbrush/) | A[`Brush`](../aspose.psd/brush/)変換機能付き. |
