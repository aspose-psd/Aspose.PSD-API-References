## ThumbnailResource.JpegOptions property

JPEG オプションを取得または設定します。サムネイル リソースが JPEG ファイル形式のみで保存される場合に適しています。このオプションは、RAW 形式が定義されている場合は効果がありません。

```csharp
public JpegOptions JpegOptions { get; set; }
```

### プロパティ値

JPEG オプション。
