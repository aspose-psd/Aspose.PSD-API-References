名前空間には、入力または出力データ フローに適したさまざまなストリーム ソースが含まれます。

## クラス

| クラス | 説明 |
| --- | --- |
| [FileCreateSource](./filecreatesource/) | 作成するファイルソースを表します. |
| [FileOpenSource](./fileopensource/) | 開くためのファイル ソースを表します。 |
| [FileSource](./filesource/) | ファイル操作が可能なファイル ソースを表します。 |
| [StreamSource](./streamsource/) | ストリーム ソースを表します。 |
