## PsdImage.HasTransparencyData property

レイヤー データを指定するときに、最初のアルファ チャネルに結合結果の透明度データが含まれているかどうかを示す値を取得または設定します。

```csharp
public bool HasTransparencyData { get; set; }
```

### プロパティ値

`真実`レイヤー データを指定するときに、最初のアルファ チャネルに結合結果の透明度データが含まれている場合。さもないと、`間違い` .
