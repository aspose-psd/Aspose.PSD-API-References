## PsdColorPalette.IsCompactPalette property

パレットを圧縮するかどうかを示す値を取得します。

```csharp
public bool IsCompactPalette { get; }
```

### プロパティ値

`真実`パレットを圧縮する場合。さもないと、`間違い`.

### 備考

コンパクト パレットとは、可能な場合、指定されたパレット エントリのみがイメージに含まれることを意味します。つまり、イメージがよりコンパクトになり、占有するスペースが少なくなります。 . この値を true に設定してパレット エントリを変更すると、データの移動が発生する可能性があるため、パフォーマンスが低下する可能性があるため、注意して使用してください。
