## GifOptions.MaxDiff property

最大許容ピクセル差を取得または設定します。 0 より大きい場合、非可逆圧縮が使用されます。 最適な非可逆圧縮の推奨値は 80 です。30 は非常に軽い圧縮で、200 は重い圧縮です。 圧縮アルゴリズムの制限により、損失がほとんどない場合に最適に機能します。非常に高い損失レベルでは、それほど多くの利益は得られません. 許容値の範囲は [0, 1000]. です。

```csharp
public int MaxDiff { get; set; }
```
