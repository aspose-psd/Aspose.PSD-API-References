## GifOptions.DoPaletteCorrection property

パレット補正を適用するかどうかを示す値を取得または設定します。

```csharp
public bool DoPaletteCorrection { get; set; }
```

### プロパティ値

`真実`パレット補正が適用されている場合。さもないと、`間違い` .

### 備考

パレット補正とは、画像がGIFにエクスポートされるたびに、ソース画像の色が分析され 最適なパレットを構築することを意味します（画像パレットが存在しないか、オプションで指定されていない場合）. 分析プロセスには時間がかかります.出力画像のカラー パレットが最適に一致し、視覚的に優れた結果が得られます。
