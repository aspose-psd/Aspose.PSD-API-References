名前空間には、さまざまなファイル形式のエクスポート、保存、または作成に適したクラスが含まれています。

## クラス

| クラス | 説明 |
| --- | --- |
| [BmpOptions](./bmpoptions/) | bmp ファイル形式の作成オプション。 |
| [CmxRasterizationOptions](./cmxrasterizationoptions/) | CMX エクスポーター オプション. |
| [GifOptions](./gifoptions/) | gif ファイル形式の作成オプション。 |
| [GraphicsOptions](./graphicsoptions/) | 埋め込みビットマップのグラフィック オプションを表します。 |
| [Jpeg2000Options](./jpeg2000options/) | Jpeg2000 ファイル形式オプション。 |
| [JpegOptions](./jpegoptions/) | jpeg ファイル形式作成オプション. |
| [MultiPageOptions](./multipageoptions/) | サポートされている複数ページの基本クラス format |
| [PdfOptions](./pdfoptions/) | PDF オプション。 |
| [PngOptions](./pngoptions/) | png ファイル形式の作成オプション. |
| [PsdOptions](./psdoptions/) | psd ファイル形式の作成オプション。 |
| [RdOptimizerSettings](./rdoptimizersettings/) | RD オプティマイザーの設定 class |
| [RenderResult](./renderresult/) | render の結果で情報を表します |
| [TiffOptions](./tiffoptions/) | tiff ファイル形式のオプション。 幅と高さのタグは、画像の作成時に幅と高さのパラメーターによって上書きされるため、直接指定する必要がないことに注意してください。 多くのオプションがデフォルト値を返すことに注意してください。このオプションは、タグ値として明示的に設定されます。タグが存在することを確認するには、Tags プロパティまたは対応する IsTagPresent メソッドを使用します。 |
| [TiffOptionsUtils](./tiffoptionsutils/) | tiff ファイル形式オプションのユーティリティ クラス。 |
| [VectorRasterizationOptions](./vectorrasterizationoptions/) | ベクター ラスター化オプション。 |
