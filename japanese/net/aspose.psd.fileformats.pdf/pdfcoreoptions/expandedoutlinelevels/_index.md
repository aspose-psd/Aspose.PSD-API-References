## PdfCoreOptions.ExpandedOutlineLevels property

PDF ファイルを表示するときに展開して表示するドキュメント アウトラインのレベル数を指定します。 0 - ドキュメント アウトラインは展開されません。 1 - ドキュメントの最初のレベルの項目が展開されます。 デフォルトは 0 です。

```csharp
public int ExpandedOutlineLevels { get; set; }
```
