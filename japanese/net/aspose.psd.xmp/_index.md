名前空間には、XMP 関連のヘルパー クラスとメソッドが含まれています。

## クラス

| クラス | 説明 |
| --- | --- |
| [LangAlt](./langalt/) | XMP 言語の代替を表します。 |
| [Namespaces](./namespaces/) | RDF ドキュメントで使用される名前空間が含まれます。 |
| [XmpArray](./xmparray/) | は Xmp 配列を表します[`XmpPackage`](../aspose.psd.xmp/xmppackage/) TODO: 配列には複雑なデータが含まれている可能性があります. |
| [XmpArrayHelper](./xmparrayhelper/) | RDF ロジックを処理するためのヘルパー クラス |
| [XmpElementBase](./xmpelementbase/) | 基本 xmp 要素に属性が含まれていることを表します。 |
| [XmpHeaderPi](./xmpheaderpi/) | XMP ヘッダー処理命令を表します。 |
| [XmpMeta](./xmpmeta/) | xmpmeta を表します。 Optional. この要素の目的は、RDF の他の非 XMP 使用を含む可能性のある一般的な XML テキスト内の XMP メタデータを識別することです。 |
| [XmpPackage](./xmppackage/) | XMP パッケージの基本抽象化を表す XmpPackage クラスを定義します。 |
| [XmpPackageBaseCollection](./xmppackagebasecollection/) | のコレクションを表します[`XmpPackage`](../aspose.psd.xmp/xmppackage/) . |
| [XmpPacketWrapper](./xmppacketwrapper/) | ヘッダーとトレーラーを含むシリアル化された xmp パッケージが含まれています。 |
| [XmpRdfRoot](./xmprdfroot/) | rdf:RDF 要素を表します。 単一の XMP パケットは、単一の rdf:RDF XML 要素を使用してシリアル化されます。 rdf:RDF 要素のコンテンツは、0 個以上の rdf:Description 要素のみで構成されるものとします。 |
| [XmpTrailerPi](./xmptrailerpi/) | XMP トレーラ処理命令を表します。 |
