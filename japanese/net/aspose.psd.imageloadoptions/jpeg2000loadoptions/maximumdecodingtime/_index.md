## Jpeg2000LoadOptions.MaximumDecodingTime property

秒単位の最大デコード時間を取得または設定します (このオプションは、メモリが非常に遅いマシンで使用して、非常に大きな画像 (解像度が 5500x6500 ピクセル以上) でプロセスが停止するのを防ぐことができます).

```csharp
public int MaximumDecodingTime { get; set; }
```

### プロパティ値

最大デコード時間.
