## TiffStreamReader.ThrowExceptions property

不正なデータ処理 (ストリームへの読み取りまたは書き込み) で例外がスローされるかどうかを示す値を取得または設定します。

```csharp
public bool ThrowExceptions { get; set; }
```

### プロパティ値

`真実`不正なデータ処理で例外がスローされた場合。それ以外の場合、エラー条件は黙って無視されます.
