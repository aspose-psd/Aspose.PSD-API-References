| [XmpChoise&lt;T&gt;](./xmpchoise-1/) | 値の語彙からの値を表します。 |
| [XmpGuid](./xmpguid/) | XMP グローバル一意識別子を表します。 |
| [XmpLocale](./xmplocale/) | 言語コードを表します。 |
| [XmpMimeType](./xmpmimetype/) | MIME タイプを表します。 |
