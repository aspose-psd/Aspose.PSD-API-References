## Configuration.BufferSizeHint property

すべての内部バッファーの最大許容サイズが定義されているバッファー サイズのヒントを取得または設定します。

```csharp
public static int BufferSizeHint { get; set; }
```

### プロパティ値

メガバイト単位のバッファ サイズのヒント。 正でない値は、内部バッファのメモリ制限がないことを意味します
