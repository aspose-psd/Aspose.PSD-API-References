## ScaleTransform(float, float) {#scaletransform}

```csharp
public void ScaleTransform(float sx, float sy)
```

| パラメータ | タイプ | 説明 |

| sx | Single | x 軸方向の変換をスケーリングする係数。 |
| sy | Single | 軸方向の変換をスケーリングする係数。 |
