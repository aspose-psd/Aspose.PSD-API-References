## TransparencySupporter.Opacity property

オブジェクトの不透明度を取得または設定します。値は 0 から 1 の間である必要があります。値 0 はオブジェクトが完全に表示されることを意味し、値 1 はオブジェクトが完全に不透明であることを意味します。

```csharp
public float Opacity { get; set; }
```

### プロパティ値

不透明度の値。
