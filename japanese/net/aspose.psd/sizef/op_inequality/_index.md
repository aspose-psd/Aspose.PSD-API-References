## SizeF Inequality operator

2 つの[`SizeF`](../)構造が異なります.

```csharp
public static bool operator !=(SizeF size1, SizeF size2)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| size1 | SizeF | の[`SizeF`](../)不等式演算子の左側の構造。 |
| size2 | SizeF | の[`SizeF`](../)不等式演算子の右側の構造。 |

### 戻り値

この演算子は、次の場合に true を返します。*size1*と*size2*幅または高さが異なります。偽の場合*size1*と*size2*は同じ。
