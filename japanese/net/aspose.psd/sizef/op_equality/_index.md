## SizeF Equality operator

2 つの[`SizeF`](../)構造は等しい.

```csharp
public static bool operator ==(SizeF size1, SizeF size2)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| size1 | SizeF | の[`SizeF`](../)等号演算子の左側の構造。 |
| size2 | SizeF | の[`SizeF`](../)等号演算子の右側の構造。 |

### 戻り値

この演算子は、次の場合に true を返します。*size1*と*size2*幅と高さが等しい。それ以外の場合は false。
