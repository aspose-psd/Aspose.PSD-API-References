## RotateFlipType enumeration

```csharp
public enum RotateFlipType
```

| Rotate270FlipNone | `3` | 反転せずに時計回りに 270 度の回転を指定します。 |

| Rotate180FlipX | `6` | 時計回りに 180 度回転した後、水平方向に反転することを指定します。 |
| Rotate270FlipX | `7` | 時計回りに 270 度回転した後、水平方向に反転することを指定します。 |
| RotateNoneFlipY | `8` | 時計回りに回転せず、垂直方向に反転することを指定します。 |

| Rotate270FlipXY | `15` | 時計回りに 270 度回転した後、水平方向と垂直方向に反転することを指定します。 |
