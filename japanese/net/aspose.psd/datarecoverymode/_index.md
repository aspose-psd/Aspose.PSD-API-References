## DataRecoveryMode enumeration

データ復旧モード.

```csharp
public enum DataRecoveryMode
```

### 値

| 名前 | 価値 | 説明 |
| --- | --- | --- |
| None | `0` | データの回復は暗示されません。ファイル形式に破損したデータがある場合は常に、適切な例外がスローされます. |
| ConsistentRecover | `1` | コンシステント リカバリ モードでは、破損によってファイル フォーマットが壊れず、さらに適切な処理が可能である限り、すべてのデータのリカバリが試行されます。 |
| MaximalRecover | `2` | 最大回復モードでは、ファイル形式の構造が壊れていてもすべてのデータが回復され、さらに処理を行うと予期せぬ結果が生じる可能性があります。 |
