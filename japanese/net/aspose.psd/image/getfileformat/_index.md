## GetFileFormat(string) {#getfileformat_1}

ファイル形式を取得します。

```csharp
public static FileFormat GetFileFormat(string filePath)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| filePath | String | ファイルパス. |

### 戻り値

決定されたファイル形式.

### 備考

決定されたファイル形式は、指定されたイメージをロードできることを意味するものではありません。 CanLoad メソッドのオーバーロードの 1 つを使用して、ファイルをロードできるかどうかを判断します。
