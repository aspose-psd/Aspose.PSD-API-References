## RectangleF Equality operator

2 つの[`RectangleF`](../)構造体の位置とサイズは同じです.

```csharp
public static bool operator ==(RectangleF left, RectangleF right)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| left | RectangleF | の[`RectangleF`](../)等値演算子の左側にある構造。 |
| right | RectangleF | の[`RectangleF`](../)等値演算子の右側にある構造。 |

### 戻り値

この演算子は、指定された 2 つの場合に true を返します。[`RectangleF`](../)構造は等しい[`X`](../x/) 、[`Y`](../y/) 、[`Width`](../width/)、 と[`Height`](../height/)プロパティ。
