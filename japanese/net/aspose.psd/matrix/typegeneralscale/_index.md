## Matrix.TypeGeneralScale field

一般的なスケールは、垂直ベクトル間の角度 を変更せずに、ベクトルの長さに異なる 量を x 方向と y 方向に乗算します. このフラグ ビットは、TypeUniformScale フラグと相互に排他的です.

```csharp
public const int TypeGeneralScale;
```
