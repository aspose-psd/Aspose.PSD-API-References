## Matrix.TypeIdentity field

恒等変換は、出力座標が 常に入力座標と同じである変換です. この変換が恒等変換以外の場合、 タイプは定数 GENERAL_TRANSFORM または適切なフラグ ビットの 組み合わせのいずれかになります。この変換が実行するさまざまな座標 変換.

```csharp
public const int TypeIdentity;
```
