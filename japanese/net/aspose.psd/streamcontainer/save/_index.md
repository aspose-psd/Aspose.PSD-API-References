## Save(Stream, int, long) {#save_2}


| パラメータ | タイプ | 説明 |

| bufferSize | Int32 | バッファサイズ。デフォルトでは[`ReadWriteBytesCount`](../readwritebytescount/)値が使用されます。 |
| length | Int64 | コピーするストリーム データの長さ。デフォルトでは、長さは[`Length`](../length/)価値。 |
