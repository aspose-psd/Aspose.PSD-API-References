## Point Inequality operator

2 つの比較[`Point`](../)オブジェクト。結果は、[`X`](../x/)また[`Y`](../y/)二つの性質[`Point`](../)オブジェクトが等しくありません.

```csharp
public static bool operator !=(Point point1, Point point2)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| point1 | Point | 最初の[`Point`](../)比べる。 |
| point2 | Point | 2番目[`Point`](../)比べる。 |

### 戻り値

次のいずれかの値の場合は true[`X`](../x/)プロパティまたは[`Y`](../y/)の性質*point1*と*point2*異なる;それ以外の場合は false。
