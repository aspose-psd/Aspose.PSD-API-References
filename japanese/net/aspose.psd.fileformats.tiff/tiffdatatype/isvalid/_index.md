## TiffDataType.IsValid property

タグデータが有効かどうかを示す値を取得します。有効なタグには、保存できるデータが含まれています。無効なタグは保存できません.

```csharp
public bool IsValid { get; }
```

### プロパティ値

`真実`タグデータが有効な場合。さもないと、`間違い` .
