名前空間には、いずれかのコア PSD コンポーネントによってスローされる例外が含まれています。

## クラス

| クラス | 説明 |
| --- | --- |
| [CompressorException](./compressorexception/) | コンプレッサの例外。 |
| [DataMissmatchError](./datamissmatcherror/) | データ不一致例外 class |
| [FrameworkException](./frameworkexception/) | PSD フレームワークの例外。このクラスは、すべての Aspose.PSD 例外のコア クラスです。 Aspose.PSD エンジンによってスローされた例外と他のすべての例外タイプを区別するために導入されました。 |
| [ImageCreateException](./imagecreateexception/) | イメージ作成例外。イメージの作成中に発生します。 |
| [ImageException](./imageexception/) | 画像の例外. |
| [ImageLoadException](./imageloadexception/) | 画像読み込み例外。画像の読み込み中に発生します. |
| [ImageSaveException](./imagesaveexception/) | 画像保存例外。画像の保存中に発生します. |
| [IndexOutOFRangeException](./indexoutofrangeexception/) | コンプレッサの例外。 |
| [LimitMemoryException](./limitmemoryexception/) | 制限メモリ例外。メモリ使用量を減らす必要があるときに発生します. |
| [OperationInterruptedException](./operationinterruptedexception/) | 操作が中断されたときに発生します。 |
| [RdOptimizationError](./rdoptimizationerror/) | RD 最適化エラー例外 class |
| [StreamReadException](./streamreadexception/) | ストリーム読み取り例外。不正なオフセットとバイト カウント リクエストが原因で、ストリームの読み取りが失敗したときに発生します。 |
| [XmpException](./xmpexception/) | XMP に無効な構造がある場合にスローされる例外。 |
