名前空間には、形状セグメントから結合されたさまざまな形状が含まれています.

## クラス

| クラス | 説明 |
| --- | --- |
| [ArcShape](./arcshape/) | 円弧形状を表します。 |
| [BezierShape](./beziershape/) | ベジェ スプラインを表します。 |
| [CurveShape](./curveshape/) | 曲線のスプライン形状を表します。 |
| [EllipseShape](./ellipseshape/) | 楕円形を表します。 |
| [PieShape](./pieshape/) | パイの形を表します。 |
| [PolygonShape](./polygonshape/) | 多角形を表します。 |
| [RectangleProjectedShape](./rectangleprojectedshape/) | 特定の方向に向けられた長方形に投影された形状を表します。 隣接するエッジ間で同じエッジ長と 90 度を維持しながら空間内で回転できる 4 つのポイントによって指定されます。 |
| [RectangleShape](./rectangleshape/) | 長方形を表します。 |
| [TextShape](./textshape/) | テキスト形状を表します。 |
