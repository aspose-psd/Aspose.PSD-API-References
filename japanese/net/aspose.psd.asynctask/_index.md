名前空間は AsyncTask 処理を処理します。

## クラス

| クラス | 説明 |
| --- | --- |
| [AsyncTask](./asynctask/) | 非同期タスクを作成するための静的ファクトリ クラス |
| [AsyncTaskAction](./asynctaskaction/) | 結果のない非同期タスクの長い操作デリゲート。 |
| [AsyncTaskException](./asynctaskexception/) | 非同期タスクの例外。 |
| [AsyncTaskFunc](./asynctaskfunc/) | ジェネリック型の結果を持つ非同期タスクの長い操作デリゲート。 |
| [AsyncTaskProgress](./asynctaskprogress/) | 非同期タスクの進捗情報を提供します。 |
| [CompleteCallback](./completecallback/) | タスク完了イベントを受け取るコールバック関数. |
| [ProgressCallback](./progresscallback/) | タスク進捗変更イベントを受け取るコールバック関数. |
## インターフェース

| インターフェース | 説明 |
| --- | --- |
| [IAsyncTask](./iasynctask/) | 非同期タスク. |
| [IAsyncTaskState](./iasynctaskstate/) | 非同期タスクの状態へのアクセスを提供します。 |
