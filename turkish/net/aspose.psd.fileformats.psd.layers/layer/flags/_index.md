## Layer.Flags property

Katman bayraklarını alır veya ayarlar. bit 0 = saydamlık korumalı; bit 1 = görünür; bit 2 = eski; bit 3 = 1, Photoshop 5.0 ve sonrası için, bit 4'ün yararlı bilgilere sahip olup olmadığını söyler; bit 4 = belgenin görünümüyle ilgisi olmayan piksel verileri.

```csharp
public LayerFlags Flags { get; set; }
```
