## PdfCoreOptions.ExpandedOutlineLevels property

PDF dosyası görüntülendiğinde belge anahattında kaç düzeyin genişletilmiş olarak gösterileceğini belirtir. 0 - belge anahattı genişletilmez. 1 - belgedeki birinci düzey öğeler genişletilir ve bu şekilde devam eder. Varsayılan değer 0.

public int ExpandedOutlineLevels { get; set; }
