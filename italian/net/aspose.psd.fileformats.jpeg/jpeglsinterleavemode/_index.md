## JpegLsInterleaveMode enumeration

Definisce la modalità interleave per i dati pixel multicomponente (colore).

```csharp
public enum JpegLsInterleaveMode
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | I dati vengono codificati e archiviati come componente per componente: RRRGGGBBB. |
| Line | `1` | La modalità interleave è per riga. Viene codificata una riga completa di ciascun componente prima di passare alla riga successiva. |
| Sample | `2` | I dati vengono codificati e archiviati per campione. Per le immagini a colori questo è il formato come RGBRGBRGB. |
