Lo spazio dei nomi contiene diverse forme combinate da segmenti di forma.

## Classi

| Classe | Descrizione |
| --- | --- |
| [ArcShape](./arcshape/) | Rappresenta una forma ad arco. |
| [BezierShape](./beziershape/) | Rappresenta una spline bezier. |
| [CurveShape](./curveshape/) | Rappresenta una forma spline curva. |
| [EllipseShape](./ellipseshape/) | Rappresenta una forma ellittica. |
| [PieShape](./pieshape/) | Rappresenta una forma a torta. |
| [PolygonShape](./polygonshape/) | Rappresenta una forma poligonale. |
| [RectangleProjectedShape](./rectangleprojectedshape/) | Rappresenta una forma proiettata su un rettangolo ruotato secondo un particolare orientamento. Specificato da quattro punti che possono essere ruotati nello spazio mantenendo la stessa lunghezza dei bordi e 90 gradi tra i bordi adiacenti. |
| [RectangleShape](./rectangleshape/) | Rappresenta una forma rettangolare. |
| [TextShape](./textshape/) | Rappresenta una forma di testo. |
