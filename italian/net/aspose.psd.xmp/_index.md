Lo spazio dei nomi contiene classi e metodi helper relativi a XMP.

## Classi

| Classe | Descrizione |
| --- | --- |
| [LangAlt](./langalt/) | Rappresenta l'alternativa del linguaggio XMP. |
| [Namespaces](./namespaces/) | Contiene gli spazi dei nomi utilizzati nel documento RDF. |
| [XmpArray](./xmparray/) | Rappresenta Xmp Array in[`XmpPackage`](../aspose.psd.xmp/xmppackage/) . TODO: l'array può contenere dati complessi. |
| [XmpArrayHelper](./xmparrayhelper/) | La classe helper per l'elaborazione della logica RDF |
| [XmpElementBase](./xmpelementbase/) | Rappresenta l'elemento xmp di base contenente attributi. |
| [XmpHeaderPi](./xmpheaderpi/) | Rappresenta l'istruzione di elaborazione dell'intestazione XMP. |
| [XmpMeta](./xmpmeta/) | Rappresenta xmpmeta. Facoltativo. Lo scopo di questo elemento è identificare i metadati XMP all'interno del testo XML generale che potrebbe contenere altri usi non XMP di RDF. |
| [XmpPackage](./xmppackage/) | Definisce la classe XmpPackage che rappresenta l'astrazione di base per il pacchetto XMP. |
| [XmpPackageBaseCollection](./xmppackagebasecollection/) | Rappresenta la raccolta di[`XmpPackage`](../aspose.psd.xmp/xmppackage/) . |
| [XmpPacketWrapper](./xmppacketwrapper/) | Contiene il pacchetto xmp serializzato che include intestazione e trailer. |
| [XmpRdfRoot](./xmprdfroot/) | Rappresenta l'elemento rdf:RDF. Un singolo pacchetto XMP deve essere serializzato utilizzando un singolo elemento rdf:RDF XML. Il contenuto dell'elemento rdf:RDF deve consistere solo di zero o più elementi rdf:Description. |
| [XmpTrailerPi](./xmptrailerpi/) | Rappresenta l'istruzione di elaborazione del trailer XMP. |
## Interfacce

| Interfaccia | Descrizione |
| --- | --- |
| [IXmlValue](./ixmlvalue/) | Converte i valori xmp nella rappresentazione di stringa XML. |
