Lo spazio dei nomi contiene le eccezioni generate da qualsiasi componente PSD principale.

| Classe | Descrizione |
| --- | --- |

| [ImageCreateException](./imagecreateexception/) | L'immagine crea un'eccezione. Si verifica durante la creazione dell'immagine. |

| [IndexOutOFRangeException](./indexoutofrangeexception/) | L'eccezione del compressore. |
| [LimitMemoryException](./limitmemoryexception/) | L'eccezione di memoria limite. Si verifica quando è necessario ridurre l'utilizzo della memoria. |
| [OperationInterruptedException](./operationinterruptedexception/) | Si verifica quando un'operazione viene interrotta. |
| [RdOptimizationError](./rdoptimizationerror/) | Eccezione errore ottimizzazione RD class |

| [XmpException](./xmpexception/) | L'eccezione generata quando XMP ha una struttura non valida. |
