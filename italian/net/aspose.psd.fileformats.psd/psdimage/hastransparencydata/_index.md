## PsdImage.HasTransparencyData property

Ottiene o imposta un valore che indica se il primo canale alfa contiene i dati di trasparenza per il risultato unito quando si specificano i dati dei layer.

```csharp
public bool HasTransparencyData { get; set; }
```

### Valore della proprietà

`VERO` se il primo canale alfa contiene i dati di trasparenza per il risultato unito quando si specificano i dati dei livelli; Altrimenti,`falso` .
