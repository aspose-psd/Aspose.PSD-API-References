Lo spazio dei nomi gestisce l'elaborazione del formato file Psd.

## Classi

| Classe | Descrizione |
| --- | --- |
| [PsdColorPalette](./psdcolorpalette/) | La tavolozza dei colori PSD. |
| [PsdImage](./psdimage/) | Definisce la classe PsdImage che offre la possibilità di caricare, modificare, salvare file PSD nonché aggiornare le proprietà, aggiungere filigrane, eseguire operazioni grafiche o convertire un formato di file in un altro. Aspose.PSD supporta l'importazione come livello e l'esportazione in seguenti formati: Png, Jpeg, Jpeg2000, Gif, Bmp, Tiff, Psd, Psb insieme all'esportazione in Pdf con testo selezionabile |
| [ResourceBlock](./resourceblock/) | Il blocco delle risorse. |
| [SmartObjectProvider](./smartobjectprovider/) | Definisce il fornitore di oggetti intelligenti che consente di ottenere/impostare origini dati dalle risorse di collegamento globale del file PSD e dai relativi contenuti. |
