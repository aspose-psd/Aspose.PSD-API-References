## TiffStreamReader.ThrowExceptions property

Ottiene o imposta un valore che indica se vengono generate eccezioni in caso di elaborazione dati errata (lettura o scrittura nel flusso).

```csharp
public bool ThrowExceptions { get; set; }
```

### Valore della proprietà

`VERO` se vengono lanciate eccezioni sull'elaborazione errata dei dati; in caso contrario, le condizioni di errore vengono silenziosamente ignorate.
