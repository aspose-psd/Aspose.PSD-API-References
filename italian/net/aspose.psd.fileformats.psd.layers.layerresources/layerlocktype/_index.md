## LayerLockType enumeration

Opzioni di blocco livello

```csharp
[Flags]
public enum LayerLockType
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Nessun blocco del livello |
| LockTransparentPixels | `1` | Blocca parzialmente un livello: limita la modifica alle parti opache del livello. Questa opzione equivale all'opzione Mantieni trasparenza nelle versioni precedenti di Photoshop. |
| LockImagePixels | `2` | Blocca parzialmente un livello - Impedisce la modifica dei pixel del livello utilizzando gli strumenti di pittura. |
| LockPosition | `4` | Blocca parzialmente un livello: impedisce lo spostamento dei pixel del livello. |
| LockAll | `7` | Blocca tutte le proprietà di un layer |
