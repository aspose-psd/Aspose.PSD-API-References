## LinkDataSource.AssetLockedState property

Ottiene o imposta un valore che indica se l'asset PSD è bloccato. Lo stato dell'asset bloccato, per le risorse Adobe® Photoshop® СС Libraries.

```csharp
public bool AssetLockedState { get; set; }
```

### Valore della proprietà

`VERO` se la risorsa Adobe® Photoshop® è bloccata; Altrimenti,`falso` .
