## IPlacedLayerResource.IsCustom property

Ottiene o imposta un valore che indica se lo stile warp di questa istanza è personalizzato. Se vero contiene punti mesh. Se impostato su false cancella i punti mesh.

```csharp
public bool IsCustom { get; set; }
```

### Valore della proprietà

`VERO` se la risorsa layer oggetto posizionato o smart ha uno stile personalizzato; Altrimenti,`falso` .
