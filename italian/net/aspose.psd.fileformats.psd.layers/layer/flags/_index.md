## Layer.Flags property

Ottiene o imposta i flag del livello. bit 0 = trasparenza protetta; bit 1 = visibile; bit 2 = obsoleto; bit 3 = 1 per Photoshop 5.0 e versioni successive, indica se il bit 4 contiene informazioni utili; bit 4 = dati pixel irrilevanti per l'aspetto del documento.

```csharp
public LayerFlags Flags { get; set; }
```

### Valore della proprietà

Il livello segnala.
