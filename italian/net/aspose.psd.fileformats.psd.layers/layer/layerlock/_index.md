## Layer.LayerLock property

Ottiene o imposta il blocco del livello. Si noti che se il flag LayerFlags.TransparencyProtected è impostato, verrà sovrascritto dal flag di blocco del livello. Per restituire il flag LayerFlags.TransparencyProtected è necessario applicare l'opzione layer layer.Flags &#x7C;= LayerFlags.TransparencyProtected

```csharp
public LayerLockType LayerLock { get; set; }
```

### Valore della proprietà

Il blocco del livello.
