## ImageLoadersRegistry.GetFirstSupportedDescriptorByTypeName method

Ottiene il primo descrittore supportato in base al relativo nome del tipo.

```csharp
public static IImageLoaderDescriptor GetFirstSupportedDescriptorByTypeName(
    string descriptorTypeName)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| descriptorTypeName | String | Il nome del tipo di descrittore. |

### Valore di ritorno

Il primo descrittore del caricatore trovato o null se non viene trovato tale descrittore.

### Osservazioni

Il primo descrittore del caricatore sarà effettivamente l'ultimo registrato.
