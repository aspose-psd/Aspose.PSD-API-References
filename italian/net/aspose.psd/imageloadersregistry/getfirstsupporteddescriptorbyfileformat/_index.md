## ImageLoadersRegistry.GetFirstSupportedDescriptorByFileFormat method

Ottiene il primo formato di file supportato in base al nome del tipo.

```csharp
public static IImageLoaderDescriptor GetFirstSupportedDescriptorByFileFormat(FileFormat fileFormat)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| fileFormat | FileFormat | Il formato di file descrittore supportato. |

### Valore di ritorno

Il primo descrittore del caricatore trovato o null se non viene trovato tale descrittore.

### Osservazioni

Il primo descrittore del caricatore sarà effettivamente l'ultimo registrato.
