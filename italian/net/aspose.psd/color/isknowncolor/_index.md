## Color.IsKnownColor property

Ottiene un valore che indica se this[`Color`](../) struttura è un colore predefinito. I colori predefiniti sono rappresentati dagli elementi del[`KnownColor`](../../knowncolor/) enumerazione.

```csharp
public bool IsKnownColor { get; }
```

### Valore della proprietà

Vero se questo[`Color`](../) è stato creato da un colore predefinito utilizzando il file[`FromName`](../fromname/) metodo o il[`FromKnownColor`](../fromknowncolor/) metodo; altrimenti, falso.
