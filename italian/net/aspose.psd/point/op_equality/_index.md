## Point Equality operator

Confronta due[`Point`](../) oggetti. Il risultato specifica se i valori di[`X`](../x/) E[`Y`](../y/) proprietà dei due[`Point`](../) gli oggetti sono uguali.

```csharp
public static bool operator ==(Point point1, Point point2)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| point1 | Point | Un primo[`Point`](../) per confrontare. |
| point2 | Point | Un secondo[`Point`](../) per confrontare. |

### Valore di ritorno

Vero se il[`X`](../x/) E[`Y`](../y/) valori di*point1* E*point2* sono uguali; altrimenti, falso.
