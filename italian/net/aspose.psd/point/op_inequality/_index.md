## Point Inequality operator

Confronta due[`Point`](../) oggetti. Il risultato specifica se i valori di[`X`](../x/) O[`Y`](../y/) proprietà dei due[`Point`](../) gli oggetti sono disuguali.

```csharp
public static bool operator !=(Point point1, Point point2)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| point1 | Point | Un primo[`Point`](../) per confrontare. |
| point2 | Point | Un secondo[`Point`](../) per confrontare. |

### Valore di ritorno

True se i valori di the[`X`](../x/) proprietà o il[`Y`](../y/) proprietà di*point1* E*point2* differire; altrimenti, falso.
