## LineCap enumeration

Specifica gli stili di berretto disponibili con cui a[`Pen`](../pen/) l'oggetto può terminare una riga.

```csharp
public enum LineCap
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Flat | `0` | Specifica un limite di linea piatta. |
| Square | `1` | Specifica un limite di linea quadrata. |
| Round | `2` | Specifica un limite di linea arrotondato. |
| Triangle | `3` | Specifica un limite di linea triangolare. |
| NoAnchor | `16` | Non specifica alcun ancoraggio. |
| SquareAnchor | `17` | Specifica un'estremità quadrata della linea di ancoraggio. |
| RoundAnchor | `18` | Specifica un tappo di ancoraggio rotondo. |
| DiamondAnchor | `19` | Specifica un tappo di ancoraggio a diamante. |
| ArrowAnchor | `20` | Specifica un tappo di ancoraggio a forma di freccia. |
| AnchorMask | `240` | Specifica un limite di riga personalizzato. |
| Custom | `255` | Specifica una maschera utilizzata per verificare se un'estremità di linea è un'estremità di ancoraggio. |
