## GraphicsUnit enumeration

Specifica l'unità di misura per i dati forniti.

```csharp
public enum GraphicsUnit
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| World | `0` | Specifica l'unità del sistema di coordinate universali come unità di misura. |
| Display | `1` | Specifica l'unità di misura del dispositivo di visualizzazione. In genere pixel per display video e 1/100 di pollice per stampanti. |
| Pixel | `2` | Specifica un pixel del dispositivo come unità di misura. |
| Point | `3` | Specifica il punto di una stampante (1/72 di pollice) come unità di misura. |
| Inch | `4` | Specifica il pollice come unità di misura. |
| Document | `5` | Specifica l'unità del documento (1/300 di pollice) come unità di misura. |
| Millimeter | `6` | Specifica il millimetro come unità di misura. |
