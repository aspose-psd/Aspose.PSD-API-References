## StringDigitSubstitute enumeration

L'enumerazione specifica come sostituire le cifre in una stringa in base alle impostazioni locali o alla lingua di un utente.

```csharp
public enum StringDigitSubstitute
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| User | `0` | Specifica uno schema di sostituzione definito dall'utente. |
| None | `1` | Specifica di disabilitare le sostituzioni. |
| National | `2` | Specifica le cifre sostitutive che corrispondono alla lingua nazionale ufficiale della località dell'utente. |
| Traditional | `3` | Specifica le cifre di sostituzione che corrispondono alla scrittura o alla lingua nativa dell'utente, che potrebbe essere diversa dalla lingua nazionale ufficiale della località dell'utente. |
