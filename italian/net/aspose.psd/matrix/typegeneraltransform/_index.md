## Matrix.TypeGeneralTransform field

Questa costante indica che la trasformazione definita da questo oggetto esegue una conversione arbitraria delle coordinate di input. Se questa trasformazione può essere classificata da una qualsiasi delle costanti di cui sopra, il tipo sarà la costante TypeIdentity o una combinazione del flag appropriato bit per le varie conversioni di coordinate eseguite da questa trasformazione.

```csharp
public const int TypeGeneralTransform;
```
