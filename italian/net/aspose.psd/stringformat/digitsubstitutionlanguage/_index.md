## StringFormat.DigitSubstitutionLanguage property

Ottiene o imposta la lingua utilizzata quando le cifre locali vengono sostituite con le cifre occidentali.

```csharp
public int DigitSubstitutionLanguage { get; set; }
```

### Valore della proprietà

Un identificatore di lingua NLS (National Language Support) che identifica la lingua che verrà utilizzata quando le cifre locali vengono sostituite con le cifre occidentali. Puoi passare ilLCID proprietà di ACultureInfo object come identificatore di lingua NLS. Ad esempio, supponi di creare aCultureInfo oggetto passando la stringa "ar-EG" a aCultureInfo constructor. Se passi il fileLCID proprietà di ciòCultureInfo oggetto insieme a. Traditional alStringDigitSubstitute) method, allora le cifre arabo-indiche verranno sostituite con le cifre occidentali al momento della visualizzazione.

### Osservazioni

Viene introdotto il setter per il metodo obsoleto SetDigitSubstitution.
