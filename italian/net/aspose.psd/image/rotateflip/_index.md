### Esempi

Questo esempio mostra l'uso dell'operazione Ruota su un'immagine. Esempio carica un file immagine esistente da una posizione su disco ed esegue l'operazione Ruota sull'immagine in base al valore di Enum Aspose.PSD.RotateFlipType

```csharp
[C#]

//Crea un'istanza della classe immagine e inizializzala con un file immagine esistente tramite Percorso file
using (Aspose.PSD.Image image = Aspose.PSD.Image.Load(@"C:\temp\image.psd"))
{
    //Ruota l'immagine di 180 gradi sull'asse X
    image.RotateFlip(Aspose.PSD.RotateFlipType.Rotate180FlipX);

    // salva tutte le modifiche.
    image.Save();
}
```
