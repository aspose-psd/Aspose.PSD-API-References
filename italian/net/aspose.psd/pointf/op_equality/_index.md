## PointF Equality operator

Confronta due[`PointF`](../) strutture. Il risultato specifica se i valori di[`X`](../x/) E[`Y`](../y/) proprietà dei due[`PointF`](../) le strutture sono uguali.

```csharp
public static bool operator ==(PointF point1, PointF point2)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| point1 | PointF | Un primo[`PointF`](../) per confrontare. |
| point2 | PointF | Un secondo[`PointF`](../) per confrontare. |

### Valore di ritorno

Vero se il[`X`](../x/) E[`Y`](../y/) valori del primo e del secondo[`PointF`](../) le strutture sono uguali; altrimenti, falso.
