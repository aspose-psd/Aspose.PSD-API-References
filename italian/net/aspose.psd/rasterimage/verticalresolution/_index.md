## RasterImage.VerticalResolution property

Ottiene o imposta la risoluzione verticale, in pixel per pollice, di this[`RasterImage`](../) .

```csharp
public virtual double VerticalResolution { get; set; }
```

### Valore della proprietà

La risoluzione verticale.

### Osservazioni

Si noti che per impostazione predefinita questo valore è sempre 96 poiché piattaforme diverse non possono restituire la risoluzione dello schermo. È possibile prendere in considerazione l'utilizzo del metodo SetResolution per aggiornare entrambi i valori di risoluzione in un'unica chiamata.
