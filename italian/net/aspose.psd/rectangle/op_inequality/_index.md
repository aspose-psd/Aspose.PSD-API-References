## Rectangle Inequality operator

Verifica se due[`Rectangle`](../) le strutture differiscono per posizione o dimensioni.

```csharp
public static bool operator !=(Rectangle left, Rectangle right)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| left | Rectangle | IL[`Rectangle`](../) struttura che si trova a sinistra dell'operatore di disuguaglianza. |
| right | Rectangle | IL[`Rectangle`](../) struttura che si trova a destra dell'operatore di disuguaglianza. |

### Valore di ritorno

Questo operatore restituisce true se uno dei[`X`](../x/) ,[`Y`](../y/) ,[`Width`](../width/) O[`Height`](../height/) proprietà dei due[`Rectangle`](../) le strutture sono disuguali; altrimenti falso.
