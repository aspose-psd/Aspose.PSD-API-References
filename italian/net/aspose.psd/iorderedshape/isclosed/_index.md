## IOrderedShape.IsClosed property

Ottiene o imposta un valore che indica se la forma ordinata è chiusa. Quando si elabora una forma ordinata chiusa, i punti iniziale e finale non hanno significato.

```csharp
public bool IsClosed { get; set; }
```

### Valore della proprietà

`VERO` se questa forma ordinata è chiusa; Altrimenti,`falso`.
