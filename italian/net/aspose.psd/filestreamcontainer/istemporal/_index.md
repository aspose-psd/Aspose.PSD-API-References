## FileStreamContainer.IsTemporal property

Ottiene o imposta un valore che indica se il flusso è temporale.

```csharp
public bool IsTemporal { get; set; }
```

### Valore della proprietà

`VERO` se il flusso è temporale; Altrimenti,`falso` .

### Osservazioni

Un flusso temporale si rimuoverà quando sarà eliminato. Se il flusso è basato sulla memoria, questa proprietà non ha alcun effetto. Il flusso può essere contrassegnato come temporale o persistente nel caso in cui sia stato creato in modo esplicito, altrimenti viene generata l'eccezione appropriata.
