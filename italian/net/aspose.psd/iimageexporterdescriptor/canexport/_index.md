## IImageExporterDescriptor.CanExport method

Determina se l'esportatore di immagini può esportare l'immagine specificata nel formato immagine specificato specificato dalle opzioni di salvataggio.

```csharp
public bool CanExport(Image image, ImageOptionsBase optionsBase)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| image | Image | L'immagine da esportare. |
| optionsBase | ImageOptionsBase | La base delle opzioni. |

### Valore di ritorno

`VERO` se l'esportatore creato da questo descrittore può esportare l'immagine specificata nel formato di file specificato; Altrimenti,`falso` .
