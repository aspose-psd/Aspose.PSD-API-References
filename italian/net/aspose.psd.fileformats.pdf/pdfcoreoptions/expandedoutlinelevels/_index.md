## PdfCoreOptions.ExpandedOutlineLevels property

Specifica quanti livelli nella struttura del documento mostrare espansi quando viene visualizzato il file PDF. 0 - la struttura del documento non è espansa. 1 - gli elementi di primo livello nel documento sono espansi e così via. Il valore predefinito è 0.

public int ExpandedOutlineLevels { get; set; }
