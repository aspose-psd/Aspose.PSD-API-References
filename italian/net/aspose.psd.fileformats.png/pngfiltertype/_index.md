## PngFilterType enumeration

Il tipo di filtro png.

```csharp
public enum PngFilterType
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Il filtro nullo, significa nessun filtro per le righe di dati dell'immagine. |
| Sub | `1` | Il filtro secondario, significa che il filtro sottrattivo verrà applicato ai dati dell'immagine. |
| Up | `2` | Il filtro up indica che verrà applicato il filtro di sottrazione riga per riga. |
| Avg | `3` | Il filtro medio, significa che quel filtro medio verrà applicato ai dati dell'immagine. |
| Paeth | `4` | Il filtro predittore paeth. |
| Adaptive | `5` | Filtro adattivo, significa che il processo di salvataggio sceglierà il filtro più adatto per ogni riga di dati. Migliore compressione, tempo di esecuzione più lento. |
