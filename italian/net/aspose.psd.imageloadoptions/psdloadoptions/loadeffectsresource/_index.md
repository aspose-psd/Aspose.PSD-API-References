## PsdLoadOptions.LoadEffectsResource property

Ottiene o imposta un valore che indica se [carica la risorsa effetti] (per impostazione predefinita la risorsa non è caricata). Quando è impostata questa opzione, solo gli effetti supportati verranno renderizzati all'immagine unita finale.

```csharp
public bool LoadEffectsResource { get; set; }
```

### Valore della proprietà

`VERO` se [risorsa effetti di carico]; altrimenti,`falso` .
