## Jpeg2000LoadOptions.MaximumDecodingTime property

Ottiene o imposta il tempo massimo di decodifica in secondi (questa opzione può essere utilizzata su computer molto lenti con memoria per evitare il blocco del processo su immagini molto grandi - risoluzione superiore a 5500x6500 pixel).

```csharp
public int MaximumDecodingTime { get; set; }
```

### Valore della proprietà

Il tempo massimo di decodifica.
