Il namespace contiene classi adatte per l'esportazione, il salvataggio o la creazione di diversi formati di file.

## Classi

| Classe | Descrizione |
| --- | --- |
| [BmpOptions](./bmpoptions/) | Le opzioni di creazione del formato file bmp. |
| [CmxRasterizationOptions](./cmxrasterizationoptions/) | le opzioni dell'esportatore CMX. |
| [GifOptions](./gifoptions/) | Le opzioni di creazione del formato di file gif. |
| [GraphicsOptions](./graphicsoptions/) | Rappresenta le opzioni grafiche per la bitmap incorporata. |
| [Jpeg2000Options](./jpeg2000options/) | Le opzioni del formato file Jpeg2000. |
| [JpegOptions](./jpegoptions/) | Il formato file jpeg crea opzioni. |
| [MultiPageOptions](./multipageoptions/) | Classe base per formati supportati da più pagine |
| [PdfOptions](./pdfoptions/) | Le opzioni PDF. |
| [PngOptions](./pngoptions/) | Il formato di file png crea opzioni. |
| [PsdOptions](./psdoptions/) | Il formato di file psd crea opzioni. |
| [RdOptimizerSettings](./rdoptimizersettings/) | Classi di impostazioni dell'ottimizzatore RD |
| [RenderResult](./renderresult/) | Rappresenta le informazioni con i risultati del rendering |
| [TiffOptions](./tiffoptions/) | Le opzioni del formato del file tiff. Nota che i tag di larghezza e altezza verranno sovrascritti durante la creazione dell'immagine dai parametri di larghezza e altezza, quindi non è necessario specificarli direttamente. Nota che molte opzioni restituiscono un valore predefinito ma ciò non significa che questa opzione è impostata esplicitamente come valore di tag. Per verificare la presenza del tag utilizzare la proprietà Tags o il metodo IsTagPresent corrispondente. |
| [TiffOptionsUtils](./tiffoptionsutils/) | Classe di utilità per le opzioni del formato file tiff. |
| [VectorRasterizationOptions](./vectorrasterizationoptions/) | Le opzioni di rasterizzazione vettoriale. |
