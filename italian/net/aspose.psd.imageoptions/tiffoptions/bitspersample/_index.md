## TiffOptions.BitsPerSample property

Ottiene o imposta i bit per campione.

```csharp
public ushort[] BitsPerSample { get; set; }
```

### Valore della proprietà

I bit per valore campione.

### Osservazioni

Quando si imposta questo valore, tenere presente che imposterà anche il valore SamplesPerPixel sulla lunghezza dell'array. Queste 2 proprietà sono strettamente collegate, quindi possono essere impostate solo tutte insieme.
