Lo spazio dei nomi fornisce classi e metodi di supporto per lavorare con diversi tipi di pennello.

## Classi

| Classe | Descrizione |
| --- | --- |
| [HatchBrush](./hatchbrush/) | Definisce un pennello rettangolare con uno stile di tratteggio, un colore di primo piano e un colore di sfondo. Questa classe non può essere ereditata. |
| [LinearGradientBrush](./lineargradientbrush/) | Incapsula a[`Brush`](../aspose.psd/brush/) con un gradiente lineare. Questa classe non può essere ereditata. |
| [LinearGradientBrushBase](./lineargradientbrushbase/) | Rappresenta a[`Brush`](../aspose.psd/brush/) con capacità di gradiente e proprietà appropriate. |
| [LinearMulticolorGradientBrush](./linearmulticolorgradientbrush/) | Rappresenta a[`Brush`](../aspose.psd/brush/) con gradiente lineare definito da più colori e posizioni appropriate. Questa classe non può essere ereditata. |
| [PathGradientBrush](./pathgradientbrush/) | Incapsula a[`Brush`](../aspose.psd/brush/) oggetto con un gradiente. Questa classe non può essere ereditata. |
| [PathGradientBrushBase](./pathgradientbrushbase/) | Rappresenta a[`Brush`](../aspose.psd/brush/) con funzionalità del gradiente del percorso di base. |
| [PathMulticolorGradientBrush](./pathmulticolorgradientbrush/) | Incapsula a[`Brush`](../aspose.psd/brush/) oggetto con un gradiente. Questa classe non può essere ereditata. |
| [SolidBrush](./solidbrush/) | Il pennello solido è destinato al disegno continuo con un colore specifico. Questa classe non può essere ereditata. |
| [TextureBrush](./texturebrush/) | Ogni proprietà di[`TextureBrush`](../aspose.psd.brushes/texturebrush/) la classe è a[`Brush`](../aspose.psd/brush/) oggetto che utilizza un'immagine per riempire l'interno di una forma. Questa classe non può essere ereditata. |
| [TransformBrush](./transformbrush/) | A[`Brush`](../aspose.psd/brush/) con capacità di trasformazione. |
